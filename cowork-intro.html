💡 비유로 이해하기

Chat은 카페에서 대화하는 것입니다. 상대가 아무리 똑똑해도 내 노트북은 볼 수 없습니다.

Cowork는 Claude를 옆자리에 앉힌 것입니다. 내 화면을 함께 보면서 직접 파일을 열고, 이메일을 확인하고, 캘린더를 정리합니다.

Code는 Claude에게 개발자 권한을 준 것입니다. 터미널을 열고 코드를 직접 짜고 실행합니다.

🤔 왜 "Cowork"라고 부를까요?

Co-worker에서 온 이름입니다. Claude가 대화 상대에 머무는 게 아니라, 내 파일과 서비스 위에서 직접 일을 처리하는 동료가 된다는 뜻입니다. 함께(Co) 일한다(Work)는 개념 그대로입니다.

Cowork로 할 수 있는 일

Claude가 직접 행동하는 6가지 대표 업무 유형

📁

파일 정리 및 자동화

다운로드 폴더 정리, 파일명 일괄 변경, 중복 파일 제거를 한 번에 처리합니다.

"이 폴더 날짜별로 정리해줘"
📊

데이터 변환

스크린샷·영수증·PDF 속 숫자를 스프레드시트로 변환합니다. 손으로 옮길 필요가 없습니다.

"이 영수증 사진들 엑셀로 만들어줘"
📝

문서 및 보고서 생성

흩어진 메모와 자료를 읽고 보고서, 제안서, PPT 초안을 자동으로 작성합니다.

"회의 노트로 보고서 초안 써줘"
📧

이메일 관리

Gmail을 직접 열어 중요 메일 분류, 뉴스레터 요약, 답장 초안을 작성합니다.

"이번 주 읽지 않은 메일 요약해줘"
📅

일정 및 미팅 정리

캘린더를 확인해 주간 브리핑 작성, 회의록 정리, 일정 충돌 확인을 합니다.

"내일 일정 정리해서 알려줘"
🔍

리서치 자동화

웹 브라우저를 직접 제어해 여러 페이지를 탐색하고 정보를 수집·정리합니다.

"경쟁사 가격 조사해서 표로 만들어줘"

한 줄 요약

Chat은 "물어보는" 도구, Cowork는 "시키는" 도구, Code는 "개발자처럼 일하는" 도구입니다. 파일·서비스가 필요하면 Cowork, 코드·자동화가 필요하면 Code를 쓰세요.

코워크 환경 설정하기

Claude Desktop에서 서비스를 연결하면 바로 사용할 수 있습니다

💻 STEP 1 Claude Desktop 설치
Claude Desktop 다운로드 ↗
  1. 오른쪽 버튼을 클릭하여 Claude Desktop을 다운로드합니다
  2. 설치 파일을 실행하고 안내에 따라 설치를 완료합니다
  3. Claude 계정으로 로그인하면 준비 완료!
📁 STEP 2 폴더 연결
  1. 채팅 입력창 하단의 + 버튼 클릭
  2. Add folder 선택
  3. 작업할 폴더를 선택하면 바로 연결 완료

💡 연결된 폴더의 파일을 읽고, 만들고, 수정할 수 있습니다

🔗 STEP 3 Google 서비스 연결
📧 Gmail 📅 Calendar 💾 Drive
  1. Claude Desktop 왼쪽 사이드바에서 사용자 정의 클릭
  2. 이어서 커넥터 탭 클릭
  3. 원하는 Google 서비스의 연결 버튼 클릭
  4. Google 계정으로 로그인 후 권한 허용
