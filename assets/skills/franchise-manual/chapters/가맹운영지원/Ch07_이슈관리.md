영업팀 가맹점 운영 지원 매뉴얼

# 가맹점 통합 이슈 관리 표준 SOP

영업팀 SV(슈퍼바이저)가 담당 가맹점에서 발생하는 매출 부진(이익률 5% 미만), QSC D등급, 컴플레인 월 3건 누적, 노무 분쟁, 법무·계약 분쟁, 폐점·재계약·양도양수 6대 이슈를 D+1 1차 회신, D+7 진단 보고, D+30 개선 과제 합의, D-90 폐점·재계약 검토, D-180 재계약 사전 진단 표준 사이클로 통합 관리하여 동일 이슈 6개월 재발 0건과 가맹점주 자영업 이탈률 5% 이내를 정량 운영한다.

## [매뉴얼 목적]

### 제 1 조 (목적)

제1조. 목적: 본 매뉴얼은 프랜차이즈 브랜드 가맹점 OPEN 안정화(D+30) 이후부터 폐점·재계약·양도양수 종결 시점까지 라이프사이클 전 구간에서 영업팀 SV(슈퍼바이저) 주도로 발생하는 6대 이슈(① 매출 부진 이익률 5% 미만 · ② QSC D등급 · ③ 컴플레인 월 3건 누적 · ④ 노무 분쟁 · ⑤ 법무·계약 분쟁 · ⑥ 폐점·재계약·양도양수)의 인지부터 1차 진단, 부서 협력 진단(인사·재무·R&D), 개선 과제 합의·이행 추적, 노무·법무 분쟁 대응, 폐점·재계약·양도양수 종결, 이슈 데이터 누적·차기 예방까지 6단계 표준 절차를 통일하는 것을 목적으로 한다. 본 절차에 의거하여 SV는 이슈 인지 D+1 1차 회신 100%, D+7 진단 보고 100%, D+30 개선 과제 합의 90% 이상, 폐점·재계약 D-90 검토 시작 100%, 재계약 사전 진단 D-180 시작, 동일 이슈 6개월 재발 0건의 6대 KPI를 정량 운영하며, Ch01 OPEN 프로세스(D+30 안정화 종결 후 본 챕터로 인계), Ch02 SV 정기 슈퍼바이징(KPI R등급 트리거), Ch03 QSC 점검(D등급 매장 인계), Ch04 교육 지원(재교육 처방 연계), Ch05 판촉 지원(매출 부진 회복 처방 연계), Ch06 시스템 지원(권한·계약 데이터 동기) 6개 챕터에서 escalate되는 모든 이슈를 종합 수렴하는 가맹운영지원 부서의 종결 챕터로 동작한다.

### 제 2 조 (개요·운영원칙)

1호. 주기: 이슈 인지 시 즉시(D+1 1차 회신) + 부서 협력 진단 D+7 이내 + 개선 과제 D+30 합의·이행 추적 + 노무·법무 분쟁 사유 발생 D+1 외부 자문 의뢰 + 폐점·재계약·양도양수 D-90 검토 시작 / D-180 재계약 사전 진단 + 분기 1회 이슈 데이터 회고

2호. 대상: 전 가맹점(OPEN D+30 안정화 종료 매장부터) / 매출 부진 매장(이익률 5% 미만 또는 원가율 38% 이상) / QSC D등급 매장(70점 미만) / 컴플레인 누적 매장(월 3건 이상) / 노무 분쟁 발생 매장(고용노동부 진정·근로감독 통지 수신 매장) / 법무·계약 분쟁 매장(가맹점주 내용증명·소장 송달 매장) / 계약 만료 D-180 도래 매장 / 폐점 의사 표명 매장·양도양수 신청 매장

3호. 담당자: 영업팀 SV(1차 진단·실무 총괄) / 영업팀 팀장(2차 검수·개선 과제 결재) / 영업본부장(3차 결재·외부 자문 비용 결재) / 대표이사(폐점·소송·합의금 1,000만원 초과 결재) / 외부 노무사(노무 분쟁 자문) / 외부 법무사·법무법인(계약·소송 자문) / 인사팀(노무·교육 협력) / 재무팀(정산·합의금 집행) / R&D팀(QSC·메뉴 재교육 협력) / 가맹점주(이행 책임)

4호. 항목: 이슈 6대 분류 코드(M·Q·C·L·J·E), 본사 ERP [이슈관리]·[운영진단] 데이터, POS 시스템 매출·KPI 신호등, 가맹점주 ERP 손익 리포트, 카카오톡 가맹점주 단체방 협의록, 외부 노무·법무 ERP 자문 회신, 노무 진정·근로감독 통지서 사본, 가맹 계약 원본·갱신 동의서·내용증명·소장, 폐점·양도양수 정산 명세서, 합의금·환급금·보증금 정산 내역, 이슈 종결 보고서, 분기 이슈 회고 보고서

5호. 보고방식: SV → 영업팀 팀장 → 영업본부장 결재(폐점·소송·합의금 1,000만원 초과 시 대표이사 결재) / 노무 분쟁은 SV → 영업팀 팀장 → 인사팀 동시 공유 후 외부 노무사 자문 의뢰 / 법무·계약 분쟁은 SV → 영업팀 팀장 → 영업본부장 → 외부 법무사 자문 동시 진행 / 양도양수는 SV → 영업팀 팀장 → 영업본부장 → 대표이사 결재 라인 D+7 이내 마감

6호. 처리 시한: 이슈 인지 D+1 09시까지 가맹점주·영업팀 팀장 1차 회신 100%, D+7 이내 부서 협력 진단 보고서 영업본부장 결재, D+30 이내 개선 과제 합의서 90% 이상 회수, 노무 분쟁 D+1 외부 노무사 자문 의뢰, 법무·계약 분쟁 D+3 외부 법무사 자문 의뢰, 폐점 의사 표명 D-90 본부장 검토 시작, 재계약 만료 D-180 사전 진단 시작, 양도양수 신청 D+30 이내 본사 사전 동의 회신

### 제 3 조 (구성)

제3조. 본 매뉴얼은 총 6단계 업무 흐름(① 이슈 인지·1차 진단(D+1 회신) → ② 부서 협력 진단(인사·재무·R&D, D+7) → ③ 개선 과제 합의·이행 추적(D+30) → ④ 노무·법무 분쟁 대응(외부 자문) → ⑤ 폐점·재계약·양도양수(D-180 / D-90) → ⑥ 이슈 데이터 누적·차기 예방(분기 1회))으로 구성된다. 본사 ERP [이슈관리]·[운영진단], POS 시스템 매출·KPI 데이터, 가맹점주 ERP, 카카오톡 가맹점주 단체방, 외부 노무·법무 ERP, 폐점·양도양수 계약 시스템 6개 시스템을 결합 운용하며, Ch01 OPEN 프로세스·Ch02 SV 슈퍼바이징·Ch03 QSC 점검·Ch04 교육 지원·Ch05 판촉 지원·Ch06 시스템 지원 6개 챕터에서 발생한 이슈가 R등급 또는 임계값 초과로 escalate될 때 본 챕터로 자동 인계된다.

## [업무 상세작성]

### STEP 1  이슈 인지·1차 진단 (D+1 회신)

① 6대 이슈 분류 코드 부여 및 D+1 1차 회신
- SV는 Ch02 슈퍼바이징 KPI 신호등 R등급, Ch03 QSC D등급(70점 미만), 홀서비스 Ch02 컴플레인 월 3건 누적, 가맹점주 노무·법무 통지서 수신, 계약 만료 D-180 도래, 폐점·양도양수 의사 표명 6개 트리거를 인지한 즉시 6대 이슈 분류 코드 M(매출 부진)·Q(QSC)·C(컴플레인)·L(노무)·J(법무)·E(폐점·재계약·양도양수) 중 1개를 부여하여 본사 ERP [이슈관리] 티켓에 등록한다.
- 이슈 인지 시각으로부터 D+1 09시까지 가맹점주에게 카카오톡 가맹점주 단체방 또는 SV 직통 유선 1개 채널로 1차 회신을 송부하고 영업팀 팀장에게 동시 사내 채널 공유한다.
- 본사 ERP [이슈관리] 티켓에 인지 시각, 가맹점 코드, 이슈 분류 코드, 트리거 출처(Ch01~Ch06 챕터·홀서비스 Ch02·외부 통지서) 4개 항목을 D+1 12시까지 100% 입력 완료한다.
- ※ 이슈 인지 D+1 09시 1차 회신 누락 매장은 SV 월간 KPI에서 1차 회신율 항목 1건당 5점 감점 완료한다.

② 1차 진단 보고서 작성 및 트리거 데이터 첨부
- SV는 D+3 이내 1차 진단 보고서를 작성하여 본사 ERP [운영진단]에 등록하며 POS 시스템 직전 4주 매출 데이터, 본사 ERP 매입·폐기율, QSC 38개 항목 점수, 컴플레인 대장 직전 1개월 누적 4종 데이터를 자동 호출 첨부한다.
- 매출 부진(M) 코드는 이익률 5% 미만·원가율 38% 이상 2개 임계값을 동시 충족한 매장을 R등급으로 분류하여 영업팀 팀장 결재를 D+3 이내 상신한다.
- 노무(L)·법무(J)·폐점·재계약·양도양수(E) 코드는 외부 통지서·내용증명·소장 사본을 D+1 이내 인사팀·법무 담당자에게 동시 공유하여 외부 자문 의뢰 트리거를 자동 발동한다.
- ※ 1차 진단 보고서 미제출 매장이 SV 담당 매장 5% 이상 발생 시 영업본부장 직접 점검 대상으로 자동 전환 완료.

### STEP 2  부서 협력 진단 (인사·재무·R&D, D+7)

① 3개 부서 합동 진단 회의 개최
- SV는 이슈 코드별 협력 부서를 매출 부진(M)·QSC(Q) 코드는 R&D팀·재무팀, 컴플레인(C) 코드는 R&D팀·인사팀, 노무(L) 코드는 인사팀·외부 노무사, 법무(J)·폐점·재계약·양도양수(E) 코드는 재무팀·외부 법무사·법무법인 3개 라인으로 분류하여 D+5 이내 합동 진단 회의를 소집한다.
- 합동 진단 회의는 SV 주관으로 90분 이상 진행하며 가맹점주 ERP 손익 리포트, POS 시스템 시간대별 매출, 본사 ERP [QSC점검] 채점표, 컴플레인 대장, 노무·법무 통지서 5종 자료를 동시 검토한다.
- 회의록은 SV 방문일지(태블릿)에 즉시 입력하여 본사 ERP [운영진단]에 D+5 18시까지 등록하고 카카오톡 사내 채널에 공유한다.
- ※ 합동 진단 회의에 R&D팀·재무팀·인사팀 중 2개 부서 이상 불참 시 영업본부장 직접 소집으로 D+7 이내 재개최 완료.

② D+7 진단 보고서 영업본부장 결재 및 5대 KPI 신호등 작성
- SV는 진단 보고서에 이슈 원인 3건 이내, 단기·중기·장기 개선 과제 5건 이내, 부서별 책임 분장 표, 예상 합의 시한(D+30 이내) 4개 항목을 표준 양식으로 정리하여 영업팀 팀장 검수 후 D+7 18시까지 영업본부장에게 결재 상신한다.
- 5대 KPI(이익률·원가율·QSC 점수·컴플레인 누적·노무·법무 분쟁 진행률)를 색상 신호등(녹색 정상·노랑 경계·빨강 R등급) 시각화 표로 보고서에 첨부한다.
- 합의금·환급금·보증금 정산 추정액 1,000만원 초과 시 진단 보고서에 대표이사 추가 결재란을 추가하여 D+7 이내 동시 상신한다.
- ※ D+7 이내 영업본부장 결재 누락 시 다음 단계 STEP 3 개선 과제 합의가 자동 보류되므로 결재 완료 확인 필수.

### STEP 3  개선 과제 합의·이행 추적 (D+30)

① 개선 과제 5건 이내 합의서 서면화
- SV는 D+10 이내 가맹점주와 90분 이상 면담을 진행하여 STEP 2 진단 보고서의 개선 과제 5건 이내를 항목·실행 시한·책임자·검증 방법 4개 칸 표 양식으로 합의서에 서면화한다.
- 매출 부진(M) 코드 합의서는 Ch05 판촉 지원 STEP 3 매장 자체 판촉 코칭 D+14 이내 1회 편성, R&D팀 메뉴 엔지니어링 D+30 이내 1회 진단, 인건비 25% 이내·포장재 4% 이내·원가율 33% 이내 3개 임계값 회복 3건을 표준 처방으로 등록한다.
- QSC(Q) 코드 합의서는 Ch04 교육 지원 STEP 3 교육 LMS QSC·결제·위생 4과목 재교육 D+15 이내 100% 이수, R&D팀 본사 4박 5일 합숙 재교육 D+30 이내 1회 편성 2건을 표준 처방으로 등록한다.
- ※ 합의서 미서면화 매장은 D+30 이내 자동 R등급 재분류 처리되며 SV 월간 KPI에서 합의율 항목 1건당 3점 감점 완료한다.

② D+30 이내 90% 이상 합의서 회수 및 2주 후 재점검
- SV는 합의서 5건 이내 항목별 이행 진척률을 D+14 시점 50% 이상, D+30 시점 90% 이상 회수하도록 카카오톡 가맹점주 단체방 D+7·D+14·D+21·D+28 4회 리마인드를 발송한다.
- 합의서 D+30 이행률 90% 미달 매장은 SV가 영업팀 팀장에게 사유서를 상신하고 영업본부장 결재로 KPI 회복 시한을 D+60 또는 D+90으로 연장 처리한다.
- D+30 이행률 90% 이상 매장은 Ch02 SV 슈퍼바이징 STEP 5 2주 후 재점검(D+44)에 자동 편성하여 직전 동기 대비 매출 변화율, 원가율 변화, QSC 점수 변화 3개 지표를 비교한다.
- ※ 합의서 이행률 50% 미달 매장이 SV 담당 매장의 10% 이상 발생 시 영업본부장 주관 회고 회의 D+14 이내 참석 완료.

### STEP 4  노무·법무 분쟁 대응 (외부 자문)

① 노무 분쟁 D+1 외부 노무사 자문 의뢰
- SV는 가맹점주가 고용노동부 진정·근로감독 통지서·임금체불 진정서를 수신한 사실을 인지한 즉시 D+1 09시까지 인사팀 노무 담당자와 외부 노무사에게 통지서 사본·근로계약서·급여 명세서 직전 6개월·근태 기록 4종 자료를 동시 송부한다.
- 외부 노무사 자문 회신은 D+5 이내 수령하며 회신서에는 이행 의무 항목, 합의금 추정액, 합의 시한, 행정처분 위험도 4개 항목을 표준 양식으로 명기 요청한다.
- 합의금 1,000만원 이하 사안은 영업팀 팀장 결재로 처리하고 1,000만원 초과 사안은 영업본부장 검수 후 대표이사 결재 라인을 D+7 이내 상신한다.
- ※ 노무 분쟁 진행 중 매장의 4대보험·근로계약서 미정비 사실이 적발될 경우 SV가 D+3 이내 시정 요구서를 가맹점주에게 송부 완료한다.

② 법무·계약 분쟁 D+3 외부 법무사 자문 의뢰 및 분쟁 종결 보고서
- SV는 가맹점주 내용증명·소장·가맹사업거래 분쟁 조정 신청서를 수신한 즉시 D+3 이내 외부 법무사·법무법인에 가맹 계약 원본, 분쟁 통지서, 직전 3년 SV 방문일지, 카카오톡 가맹점주 단체방 협의록 4종 자료를 송부하여 자문을 의뢰한다.
- 외부 법무사 자문 회신은 D+10 이내 수령하며 답변서·조정안·합의금 한도 3개 항목을 표준 양식으로 회수하고 영업본부장 결재 라인을 D+14 이내 상신한다.
- 분쟁 종결 후 SV는 분쟁 종결 보고서에 분쟁 사유, 외부 자문 회수 비용, 합의금·환급금 집행액, 재발 방지 개선 과제 3건 4개 항목을 정리하여 본사 ERP [이슈관리]에 D+30 이내 등록 완료한다.
- ※ 합의금·환급금·보증금 정산 1,000만원 초과 시 대표이사 결재 누락은 자금 집행 보류 사유로 즉시 처리되므로 결재 라인 동시 상신 필수.

### STEP 5  폐점·재계약·양도양수 (D-180 / D-90)

① 재계약 D-180 사전 진단 및 폐점 D-90 검토 시작
- SV는 가맹 계약 만료 D-180 시점에 직전 12개월 매출, 이익률, QSC 점수, 컴플레인 누적, 본사 정책 위반 5개 항목으로 재계약 사전 진단표를 작성하여 영업팀 팀장에게 D-180 18시까지 보고한다.
- 가맹점주 폐점 의사 표명(자영업 이탈·매출 부진 누적·노무·법무 분쟁 누적 사유) 인지 D-90 시점부터 SV → 영업팀 팀장 → 영업본부장 → 대표이사 검토 라인을 자동 발동하고 폐점 정산 명세서 작성을 재무팀에 D-90 이내 의뢰한다.
- 재계약 사전 진단표 결과 매출 부진(M)·QSC(Q)·컴플레인(C) 3개 코드가 직전 12개월 R등급으로 누적된 매장은 D-180 시점에 본사 갱신 거절 검토 대상으로 분류한다.
- ※ 재계약 D-180 사전 진단 누락 매장은 가맹사업법상 갱신 거절 통지 시한(D-180) 위반 사유로 SV 월간 KPI 5점 감점 완료.

② 양도양수 D+30 본사 사전 동의 회신 및 폐점 정산 종결
- 가맹점주 양도양수 신청서 접수 시 SV는 양수인 사업자등록증·자금 출처 증빙·신원 조회 3종 자료를 D+10 이내 회수하여 영업팀 팀장 1차 검수 후 영업본부장·대표이사 결재 라인을 D+30 이내 상신한다.
- 양도양수 본사 사전 동의 회신은 D+30 이내 발급하며 양수인은 교육 LMS OPEN 패키지 4과목 100% 이수, 본사 4박 5일 합숙 재교육 1회 수료를 D+90 이내 완료해야 본사 동의 효력이 유지된다.
- 폐점 정산은 본사 ERP 잔여 식자재 회수액·POS 시스템 잔여 결제 정산·인테리어 보증금 환급·가맹비·교육비 잔여분 4개 항목을 재무팀이 D+60 이내 정산 명세서로 종결하고 SV가 매장 시설물 회수 사진 10컷을 본사 ERP [이슈관리]에 등록 완료한다.
- ※ 양도양수 본사 사전 동의 누락 상태에서 진행된 양도양수는 가맹 계약 위반 사유로 즉시 계약 해지 처리 대상이며 SV가 D+1 이내 영업본부장 보고 완료.

### STEP 6  이슈 데이터 누적·차기 예방 (분기 1회)

① 분기 이슈 회고 보고서 및 동일 이슈 6개월 재발 0건 검증
- SV는 분기 종료 D+5까지 본사 ERP [이슈관리] 누적 티켓을 6대 이슈 분류 코드(M·Q·C·L·J·E)별로 집계하여 발생 건수, 평균 종결 시한, 합의금·환급금 집행액 합계, 동일 매장 6개월 재발 건수 4개 항목으로 분기 이슈 회고 보고서를 작성한다.
- 동일 매장에서 6개월 이내 동일 코드 이슈가 재발한 건은 SV 월간 KPI에서 재발 0건 항목 1건당 5점 감점 처리하고 영업본부장 직접 점검 대상으로 자동 전환한다.
- 분기 보고서는 SV → 영업팀 팀장 → 영업본부장 결재 라인으로 분기 D+15 이내 마감하고 합의금·환급금 분기 합계 5,000만원 초과 시 대표이사 추가 결재를 D+20 이내 상신한다.
- ※ 분기 이슈 회고 보고서 미제출 SV는 분기 평가에서 이슈 관리 항목 0점 처리되며 다음 분기 부진 매장 라우팅이 자동 재배정 완료.

② Ch01~Ch06 환류 개선 및 차기 예방 과제 등록
- 분기 회고 결과 매출 부진(M) 코드 누적 상위 3개 매장은 Ch05 판촉 지원·Ch04 교육 지원 차기 분기 차분기 우선 처방 대상으로 등록하고 QSC(Q) 코드 누적 상위 3개 매장은 Ch03 QSC 점검·Ch04 교육 지원 D+30 이내 재교육 자동 편성한다.
- 컴플레인(C) 코드 누적 매장은 홀서비스 Ch02 클레임 처리 STEP 6 반복 클레임 매장 분석·재교육 절차로 인계하고 노무(L)·법무(J) 코드 누적 매장은 인사팀·법무 담당자가 차기 분기 가맹점주 노무·법무 교육 모듈에 사례로 등록한다.
- 차기 예방 과제는 Ch01 OPEN 프로세스 D+30 안정화 점검 항목 추가, Ch02 슈퍼바이징 KPI 신호등 임계값 재설정, Ch06 시스템 지원 본사 ERP [이슈관리] 자동 알림 강화 3건을 분기 D+20까지 영업본부장 결재로 확정한다.
- ※ 분기 환류 개선 과제 미등록 시 다음 분기 동일 이슈 재발률이 30% 이상 상승하므로 분기 D+20 등록 완료를 임계값으로 관리한다.
