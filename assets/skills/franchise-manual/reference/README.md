# reference/ 폴더 사용법

이 폴더에는 **Generator가 톤·구조·디자인을 학습할 기존 완성본**을 넣습니다.

## 권장 자료

1. **해산물물류_Ch01~03.pptx** (또는 PDF 버전)
   - 좌측 빨간 밴드, 노란 박스, +mj-ea 폰트 등 디자인 스펙의 원본 사례
   - Planner가 이 파일을 분석해서 design_spec.md를 채움

2. **(선택) 다른 부서의 우수 사례 매뉴얼**
   - 톤 다양성을 학습시키고 싶을 때

## 주의사항

- 파일은 PPTX, PDF, MD 어느 형식이든 OK
- Generator는 이 폴더를 read-only로만 접근
- 대용량 파일(>10MB)은 별도 저장 후 경로만 참조 권장
