# leadership-intelligence

오늘 날짜 기준 최근 30일의 글로벌·국내 리더십 트렌드를 수집해, 시니어 임원이 10분 안에 읽는 리더십 인텔리전스 뉴스레터로 만들어 주는 스킬입니다.

## 이 스킬이 하는 일

HBR, McKinsey, Deloitte 같은 글로벌 권위 매체와 SERI CEO, LG경영연구원, DBR 같은 국내 연구기관까지 20개 큐레이션 소스에서 최근 리더십·경영 트렌드를 모읍니다. 그다음 3단계 에이전트 검증을 거쳐 품질을 끌어올립니다.

- **RED 에이전트(비판적 검토)**: 수집한 항목을 신뢰도·관련성·독창성·시의성 기준으로 채점하고, 기준에 못 미치는 항목을 걸러냅니다.
- **BLACK 에이전트(종합 편집)**: 통과한 항목을 주제별로 묶어, 단순 나열이 아니라 흐름이 보이는 한 편의 뉴스레터로 엮습니다.
- **GOLD 에이전트(임원 검증)**: 가상의 시니어 임원 관점에서 전략적 가치, 시간 대비 가치, 한국 현실 반영도를 평가합니다.

세 점수의 평균이 9.5점을 넘을 때까지 최대 3라운드를 반복합니다. 비판자와 독자 관점을 작성자와 분리해 검증하기 때문에, 1인 초안에서 흔히 생기는 출처 신뢰도와 통찰의 구멍을 메울 수 있습니다.

## 언제 쓰나

다음과 같은 요청에서 발동합니다.

- "최근 리더십 트렌드 정리해줘", "이번 달 리더십 동향 브리핑"
- "임원용 뉴스레터 만들어줘", "시니어 임원 브리핑", "리더십 인텔리전스"
- "HBR·McKinsey 리더십 정리해줘", "경영 트렌드 리포트"
- `/leadership-intelligence`

다음 경우에는 이 스킬을 쓰지 않습니다.

- 리더십 용어 한 개의 단순 정의나 개념 설명
- 이미 가진 기사 한 건만 요약하거나 번역하는 일
- 특정 인물 한 명의 평판이나 이력 조회
- 리더십과 무관한 일반 경제·산업·증시 뉴스

## 입력과 출력

**입력**: 자연어 요청 한 줄이면 됩니다. 필요하면 "AI 리더십만", "조직문화 중심으로", "영어로", "간단하게" 같은 커스터마이징 조건을 덧붙일 수 있습니다.

**출력**: 빅 픽처, 2~3개 주제 섹션(각 섹션마다 임원 시사점 포함), 이달의 한 문장, 참고 소스 목록으로 구성된 리더십 인텔리전스 브리핑입니다. RED·BLACK·GOLD 3단 검증 점수와 라운드 정보도 함께 제공합니다.

## 사용법

이 스킬은 실시간 웹 검색이 필요하므로 **WebSearch 도구**가 활성화된 환경에서 동작합니다. 검색을 쓸 수 없거나 양질의 자료가 부족하면, 추측으로 채우지 않고 수집 현황과 한계를 먼저 알려 드립니다.

자연어로 다음처럼 호출합니다.

```
이번 달 글로벌·국내 리더십 트렌드 정리해서 임원용 뉴스레터로 만들어줘
```

```
/leadership-intelligence AI 리더십 중심으로, 한국 사례 꼭 넣어서
```
