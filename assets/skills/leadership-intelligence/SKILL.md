# 리더십 인텔리전스 뉴스레터 스킬

## 이 스킬의 목적

바쁜 시니어 임원(C-suite, 본부장급)이 **10분 안에** 이번 달 글로벌·국내 리더십 트렌드의 핵심을 파악할 수 있는 뉴스레터를 생성한다. 단순 정보 나열이 아니라, 검증된 인사이트와 실행 가능한 시사점을 제공하는 것이 목표다.

3단계 에이전트(RED→BLACK→GOLD)를 거치는 이유는, 1인 작성자가 만든 초안은 출처 신뢰도·통찰의 날카로움·임원 현실 적합성에서 자주 구멍이 나기 때문이다. 비판자와 독자 관점을 분리해 검증하면 합격선까지 품질이 안정적으로 올라간다.

---

## 전체 프로세스

```
[1단계] 콘텐츠 수집
        ↓
[2단계] RED 에이전트 · 비판적 검토 (각 항목 10점 만점)
        ↓  (탈락 항목 제거)
[3단계] BLACK 에이전트 · 종합 뉴스레터 초안 작성 (10점 만점)
        ↓
[4단계] GOLD 에이전트 · 시니어 임원 관점 최종 검증 (10점 만점)
        ↓
[5단계] 라운드 점수 계산 → 9.5 이상이면 통과, 미달 시 재실행 (최대 3라운드)
        ↓
[6단계] 최종 뉴스레터 출력
```

**통과 기준:** (RED + BLACK + GOLD) / 3 ≥ **9.5점** (10점 만점)
**최대 라운드:** 3회. 3회 후에도 미달 시 최고점 결과물 제공.

## 1단계: 콘텐츠 수집

### 검색 실행 방법

먼저 **오늘 날짜를 확인**한다(날짜를 모른 채 검색하면 최신성을 판단할 수 없기 때문이다). 그다음 WebSearch 도구로 아래 소스들을 **오늘 날짜 기준 최근 30일** 범위로 검색한다. 쿼리에 연도를 넣을 때는 하드코딩하지 말고 **오늘 날짜 기준 현재 연도**를 사용한다.

**검색 쿼리 패턴(현재 연도를 [올해]로 표기):**
- `site:hbr.org leadership [올해]` (최근 30일 필터)
- `site:mckinsey.com leadership insights`
- `SERI CEO 리더십 [올해]`
- `DBR 리더십 트렌드`
- `"leadership trends" site:deloitte.com`

**수집 목표:** 15~25개 항목. 아래 소스에서 골고루 수집한다.

| 우선순위 | 소스 그룹 | 목표 수집 수 |
|---------|-----------|------------|
| 1순위 | HBR, McKinsey, Deloitte, MIT Sloan | 6~8개 |
| 2순위 | 삼성 SERI CEO, LG경영연구원, DBR | 4~6개 |
| 3순위 | Fortune, WEF, Fast Company, 현대경제연구원 | 4~6개 |

**수집 시 각 항목에 기록:**
```
번호: [N]
제목: [원문 제목]
출처: [소스명]
날짜: [YYYY-MM-DD]
URL: [링크]
핵심 내용: [2~3문장 요약]
주요 키워드: [최대 3개]
언어: [한국어/영어]
```

**엣지케이스 처리:**
- 특정 소스 접근 불가 → 건너뛰고 다른 소스 2개 추가 검색
- 30일 내 최신 자료 부족 → 45일로 범위 확대 후 날짜 명시
- 15개 미만 수집 시 → 추가 검색 후 수집 이유와 함께 보고

### 환각·출처 방지 규칙 (엄수)

이 스킬의 신뢰는 출처에서 나온다. 다음 규칙을 어기면 산출물 전체가 무효다.

- **실제로 찾은 자료만 쓴다.** WebSearch로 확인하지 못한 항목은 뉴스레터에 넣지 않는다. 그럴듯해 보인다는 이유로 제목·URL·기관명을 채우지 않는다.
- **수치·인용·날짜를 지어내지 않는다.** 통계, 설문 결과, 발언 인용, 발행일은 원문에서 확인한 값만 쓴다. 확인하지 못한 수치는 "수치 미확인"으로 남기거나 문장에서 뺀다.
- **URL을 날조하지 않는다.** 실제 검색으로 얻은 링크만 기재한다. 링크를 확인하지 못하면 "기관명, 날짜" 수준으로만 표기하고 URL 칸은 비운다.
- **직접 인용은 원문 그대로만.** 실존 인물의 발언을 임의로 만들거나 의역해 따옴표로 묶지 않는다.
- **검색 불가·자료 부족은 정직하게 보고한다.** WebSearch를 쓸 수 없거나 30~45일 내 양질 자료가 목표치에 못 미치면, 추측으로 채우지 말고 수집 현황과 한계를 사용자에게 먼저 알린 뒤 진행 여부를 정한다.

## 2단계: RED 에이전트 · 비판적 검토

RED의 역할은 **엄격한 문지기**다. 품질 기준을 통과한 항목만 다음 단계로 넘긴다. 관대하게 통과시키면 뉴스레터 전체 품질이 떨어진다는 점을 명심한다.

### 평가 기준 (10점 만점)

**① 신뢰도 (3점)**
- 3점: 세계적 권위 기관, 데이터·연구 기반, 저자 명시
- 2점: 신뢰할 수 있는 출처, 근거 있음
- 1점: 출처 불분명 또는 의견 위주
- 0점: 홍보성, 광고성, 출처 없음

**② 관련성 (3점)**
- 3점: 리더십 실무 핵심 이슈, 임원 의사결정에 직접 영향
- 2점: 리더십과 관련 있으나 간접적
- 1점: 경영 일반이지만 리더십 연결 가능
- 0점: 리더십과 무관

**③ 독창성 (2점)**
- 2점: 새로운 연구결과, 기존 통념을 뒤집는 인사이트, 신선한 관점
- 1점: 알려진 내용이지만 잘 정리됨
- 0점: 뻔한 내용 반복, 이미 알려진 정보만 나열

**④ 시의성 (2점)**
- 2점: 현재 비즈니스 환경(AI 전환기, 불확실성 시대)과 직결
- 1점: 일반적으로 유효한 내용
- 0점: 시대에 뒤진 내용

### 즉시 탈락 조건 (점수와 무관)
- 홍보성·광고성 콘텐츠
- 30일(확장 시 45일) 이상 된 자료
- 리더십과 완전히 무관한 일반 경제 뉴스

## 3단계: BLACK 에이전트 · 종합 뉴스레터 편집

BLACK의 역할은 **스토리텔러**다. 통과된 항목들에서 패턴을 찾고, 개별 정보를 연결해 임원이 "아, 이런 흐름이구나"라고 깨달을 수 있는 서사를 만든다.

### 편집 원칙

1. **클러스터링:** 비슷한 주제의 항목을 묶어 섹션을 구성한다. 2~4개 섹션이 적당하다.
2. **연결성:** 글로벌 트렌드와 국내 상황을 연결하는 문장을 반드시 포함한다.
3. **실행 가능성:** 각 섹션 마지막에 "임원이 이번 주 생각해볼 것" 1문장을 추가한다.
4. **분량:** 전체 10분 독서 분량 (약 800~1200자 한국어 기준)

## 4단계: GOLD 에이전트 · 시니어 임원 관점 검증

GOLD는 **가상의 시니어 임원**이 되어 뉴스레터를 읽고 평가한다. 이 임원은 바쁘고, 이미 많은 정보를 알고 있으며, 뻔한 내용에는 곧바로 덮어버린다.

### GOLD의 페르소나
- 대기업 본부장 또는 C-suite 임원
- 매일 경영 정보를 접하는 사람
- "이건 이미 알아", "그래서 뭐가 달라?" 반응이 기본값
- 한국 비즈니스 현실을 잘 아는 사람

### 평가 기준 (10점 만점)

**① 전략적 가치 (4점):** 현재 조직 운영·의사결정에 직결되는 내용인가
- 4점: 즉시 회의에서 꺼낼 수 있는 내용
- 3점: 전략 논의에 배경으로 활용 가능
- 2점: 알면 좋은 수준
- 1점: 이미 다 아는 내용
- 0점: 현장과 동떨어진 내용

**② 시간 대비 가치 (3점):** 10분 투자 대비 얻는 것이 충분한가
- 3점: 10분이 아깝지 않다, 다음 호도 기대된다
- 2점: 읽을 만하다
- 1점: 시간이 남으면 읽겠다
- 0점: 안 읽어도 됐을 것 같다

**③ 한국 현실 반영 (3점):** 국내 맥락이 충분히 반영됐는가
- 3점: 국내 사례와 글로벌 트렌드가 잘 연결됨
- 2점: 국내 내용이 있으나 깊이 부족
- 1점: 외국 사례 위주, 국내 적용성 불명확
- 0점: 한국 맥락 전무

### GOLD가 던지는 질문들 (모두 YES면 9.5 이상)
- "이 내용을 팀장들에게 공유하고 싶은가?"
- "다음 달도 이 뉴스레터를 기다릴 것 같은가?"
- "읽고 나서 뭔가 생각이 바뀌거나 확인됐는가?"
- "글로벌과 한국이 연결되어 있는가?"
- "뻔하지 않고 날카로운 인사이트가 있는가?"

## 5단계: 라운드 점수 계산 및 판정

### 점수 계산

```
라운드 종합 점수 = (RED 총점 + BLACK 총점 + GOLD 총점) / 3
```

### 판정 및 다음 액션
