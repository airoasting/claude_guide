F&B 프랜차이즈 예비 가맹점주 상담 분석 스킬
(franchise-consult)

📝 개요
F&B 프랜차이즈 본사가 예비 가맹점주 상담 내용을 받아 자금·운영·상권·브랜드 4개 영역을 동시에 채점하고, 최종 가맹 적격성 판단(적합/보류-정보부족/보류-개선조건부/부적합)까지 자동 도출하는 5-페르소나 협업 스킬입니다. 본사 영업기획팀이 통상 2~3시간씩 소요하던 상담 후 분석을 5분 내로 단축하고, 거부권 룰과 8점 도달 로드맵으로 단순 거절이 아닌 점주 육성 가능 후보까지 분류해내는 것을 목표로 설계했습니다.

🎭 5-페르소나 구성 (본사 의사결정 실제 역할 차용)
1. 블랙 (총괄 디렉터 · black-director)
   - 4개 페르소나 평균 종합점수 산출 + 거부권 점검 + 최종 합의문 작성
2. 레드 (리스크 분석 · red-risk)
   - 자금·부채·가족동의·실패대비책 등 위험 신호 평가 (거부권 보유)
3. 골드 (현장 운영 전문가 · gold-ops)
   - 외식업 경험·직원 관리·근무 의지·체력 평가 (거부권 보유)
4. 블루 (상권 전략가 · blue-location)
   - 입지 이해도·유동인구·경쟁매장·배달권역 평가
5. 화이트 (브랜드 매니저 · white-brand)
   - 브랜드 철학 적합성·매뉴얼 준수 의지·협업 태도 평가

📊 점수 체계
- 각 페르소나 10점 만점 채점 → 평균이 종합점수
- 정보 50% 미만 누락 시 N/A 처리(추측 채점 금지)
- 거부권과 점수 별개 표기: 점수 높아도 레드·골드 거부권 발동 시 부적합
- 8점 도달 개선 로드맵 자동 생성(점주 육성 가이드)

🔧 운영 흐름
1. 본사 직원이 모바일 앱으로 상담 내용 입력 (자동 저장, 입력 완성도 실시간 표시)
2. "분석용 출력" 버튼 → 정형 마크다운 자동 생성 → 클립보드 복사
3. 클로드 채팅에 붙여넣기 → 5-페르소나 자동 채점 + 거부권 점검 + 판정
4. 8점 도달 로드맵 출력 → 본사 영업기획팀이 상담자에게 구체적 개선 안내
5. 2차 상담 시 같은 레코드 재호출 → 점수 변화(▲/▼) 자동 비교

🎯 설계 차별점
- 거부권 + 점수 병행: 평균 점수만으로 판정하면 자금이 결정적으로 부족한 케이스를 적합으로 분류하는 사고 방지
- 보류 2단계 분할: 보류-정보부족(추가 정보 확보 필요)과 보류-개선조건부(특정 조건 충족 시 적합 전환)로 분리해 후속 액션 차별화
- 정보 공백 의무 표기: 각 페르소나가 자기 영역 미확인 항목을 명시 → 추측 채점 차단 → 본사 영업 담당자의 다음 통화 질문 리스트 자동 생성
- 단순 거절 방지: 부적합 판정 케이스도 8점 도달 로드맵을 함께 제시해 점주 육성 가능 후보로 관리
