STEP 03 : PRACTICE

따라하기: 첫 스킬 체험

3단계만 따라하면 스킬의 위력을 직접 느낍니다

1

claude.ai 접속 후 아래 프롬프트를 그대로 입력

PowerPoint 스킬이 자동 감지되어 .pptx 파일이 생성됩니다.

2024년 4분기 매출 현황에 대한 5장짜리 발표자료를 만들어줘. 1장: 매출 하이라이트 (핵심 수치 요약) 2장: 제품별 매출 분석 (비교 차트) 3장: 전년 대비 성과 (증감률) 4장: 주요 성과 요인 5장: 향후 개선 방안 각 슬라이드에 차트와 그래프를 포함해서 전문적으로 만들어줘.
2

다운로드 버튼 클릭

Claude가 PowerPoint 스킬을 자동 감지하고 .pptx 파일을 생성합니다.
다운로드해서 바로 열어보세요. 실제 업무에 사용할 수 있는 수준입니다.

3

결과물을 확인하고, 수정 요청하기

"2장의 차트를 막대에서 원형으로 바꿔줘"처럼 대화로 수정할 수 있습니다. 이것이 스킬의 진짜 힘입니다.

PRO TIP

"마케팅 전략 발표자료를 만들어줘. 타겟: 경영진, 스타일: 전문적이고 간결하게"처럼 대상과 분위기를 함께 알려주면 더 좋은 결과가 나옵니다.

STEP 04 : CREATE

나만의 스킬 만들기 : 가장 쉬운 방법

기존 양식 파일을 첨부하면 Claude가 자동으로 SKILL.md를 생성

주의

파일명은 반드시 SKILL.md입니다. 대문자로 써야 합니다.
skill.md, Skill.md 등 소문자로 쓰면 Claude가 인식하지 못합니다.

STEP 1
양식 파일 준비
평소 쓰던 PPT·Word·PDF
→
STEP 2
Claude에 첨부 + 프롬프트
아래 프롬프트 복사해서 입력
→
STEP 3
SKILL.md 자동 생성!
다운로드 → zip → 업로드
첨부한 파일은 우리 회사에서 실제로 사용하는 [보고서/제안서/이메일] 양식이야. 이 양식의 구조, 형식, 톤앤매너를 분석해서 Claude Skill용 SKILL.md 파일을 만들어줘. 조건: - name과 description은 한국어로 - description에는 이 스킬이 작동할 키워드를 3개 이상 포함 - Instructions는 단계별로 구체적으로 - Output Format은 첨부 파일의 구조를 그대로 반영 - Examples는 실제 사용 상황 3가지 이상
이게 왜 쉬운가요?

SKILL.md의 구조를 몰라도 됩니다. 평소 쓰던 파일만 첨부하면 Claude가 양식을 분석해서 스킬을 자동으로 생성해줍니다. 생성된 SKILL.md를 다운로드하고, zip으로 압축한 뒤 업로드하면 끝!

만든 스킬 업로드하기

SKILL.md를 zip으로 압축한 뒤, 4단계만 따라가세요

1

파일 준비

SKILL.md 작성 후 zip으로 압축

2

설정 진입

claude.ai → 좌측 하단 이름 → Settings → Features → Skills ON

3

업로드

Upload Skill 클릭 → zip 파일 선택

4

테스트

반드시 새 대화를 열고, description에 넣었던 키워드로 요청해보세요.
예) "주간 보고서 정리해줘" → 스킬이 자동 활성화되면 성공!

주의

기존 대화에서는 스킬이 인식되지 않을 수 있습니다. 꼭 새 대화에서 테스트하세요.
활성화되지 않으면 → description에 키워드를 더 추가하거나, Skills 토글 확인!

자주 하는 실수 & 해결법

처음 만들 때 이 3가지만 주의하면 됩니다

❌ 스킬이 자동으로 작동하지 않아요 ▼

원인: description에 사용자가 실제로 말할 키워드가 빠져 있음
해결: "매출 보고서, 월간 리포트, 성과 분석을 언급할 때 사용하세요"처럼 구체적인 키워드를 여러 개 넣으세요

❌ 결과물이 매번 달라요 (일관성 없음) ▼

원인: Instructions가 모호하거나 형식 지정이 빠져 있음
해결: "불릿으로 정리", "A4 1장 이내", "섹션 4개로 구분"처럼 구체적인 형식 규칙을 명시하세요

❌ 업로드했는데 스킬 목록에 안 보여요 ▼

확인사항:
1. zip 파일 크기 8MB 이하인지
2. zip 안에 SKILL.md가 포함되어 있는지
3. 설정 → Features에서 Skills 토글이 켜져 있는지
4. Pro 이상 플랜인지 확인하세요
