도구란

도구는 Claude의 손과 발입니다

Claude에게 일을 시키면 보통 글로 답합니다. 도구를 쥐여주면 다릅니다. 파일을 직접 열어 고치고, 메일을 찾아 읽고, 코드를 실행하고, 회사 데이터와 최신 웹을 조회합니다. 말로 설명하는 Claude와 실제로 행동하는 Claude의 차이가 바로 도구입니다.

한 문장으로

도구는 Claude가 말 대신 실제로 행동하게 해주는 모든 수단입니다.

도구가 있고 없고의 차이

같은 부탁을 해도 도구 유무에 따라 결과가 달라집니다

도구 없는 Claude

  • "엑셀 정리하는 법을 알려 드릴게요"라고 글로만 답함
  • 실제 파일은 사람이 직접 열어 작업
  • 사내 자료나 최신 정보에는 닿지 못함
  • 화면 속 대화창 안에 갇힘

도구 쥔 Claude

  • 엑셀 파일을 직접 열어 정리해 저장
  • 메일함에서 필요한 메일을 찾아 읽음
  • 코드를 실행해 결과를 눈으로 확인
  • 회사 시스템과 최신 웹을 직접 조회
🤚
비유 1: 손과 발

머리만 좋은 사람은 생각만 합니다. 손발이 있어야 문서를 만들고, 전화를 걸고, 물건을 옮깁니다. 도구는 Claude에게 달아 주는 손과 발입니다.

📱
비유 2: 스마트폰 앱

갓 산 폰은 전화와 문자만 됩니다. 앱을 깔면 지도를 보고, 결제를 하고, 번역을 합니다. 도구를 붙이는 것은 Claude에 앱을 까는 일과 같습니다.

하네스가 어떻게 일할지를 정한다면,
도구는 무엇으로 일할지를 정합니다.

다섯 종류의 도구
전체 지도 Claude의 도구는 다섯 종류로 나뉩니다. 이름은 달라도 목적은 하나입니다

MCP와 커넥터
가장 헷갈리는 둘 MCP와 커넥터는 같은 것의 두 얼굴입니다. 여기만 짚으면 나머지는 쉽습니다

MCP는 규약, 커넥터는 그 규약의 포장

둘을 헷갈리는 사람이 많습니다. 정리는 간단합니다. MCP는 외부 시스템을 잇는 약속된 방식(규약)이고, 커넥터는 그 규약을 클릭 한 번으로 쓰게 만든 포장입니다. 콘센트 규격과 멀티탭의 관계와 같습니다. 규격이 있어야 멀티탭이 나오고, 사용자는 멀티탭만 꽂으면 됩니다.

MCP · 규약 (개발자 쪽)

  • 외부 시스템을 도구로 잇는 약속된 방식
  • 터미널에서 claude mcp add로 연결
  • 목록에 없는 서비스도 직접 붙임
  • 설정과 권한을 손으로 다룸

커넥터 · 포장 (누구나)

  • 같은 MCP를 메뉴에서 클릭으로 연결
  • Claude 데스크톱과 웹의 설정에서 켜고 끔
  • Gmail, 드라이브, 노션처럼 준비된 것만
  • 코드나 터미널이 필요 없음

한 표로 비교

구분MCP커넥터
정체연결 규약(프로토콜)규약을 입힌 클릭 버전
쓰는 사람개발자, 직접 연결누구나, 준비된 목록에서
연결 방법터미널에 claude mcp add설정에서 토글로 켜기
자유도어떤 서비스든 붙임제공되는 것만
비유콘센트 규격완제품 멀티탭

그래서 "MCP를 쓴다"와 "커넥터를 붙인다"는 대개 같은 일을 가리킵니다.
개발자는 MCP라 부르고, 일반 사용자는 커넥터라 부를 뿐입니다.

무엇을 고르나
상황별 선택 하고 싶은 일부터 거꾸로 고르면 빠릅니다

내 상황엔 무엇이 맞을까

하고 싶은 일고를 도구이유
코드 없이 Gmail, 드라이브, 노션을 붙이고 싶다커넥터클릭만으로 연결
사내 시스템이나 특정 API를 직접 잇고 싶다MCP원하는 서비스를 자유롭게 연결
같은 작업 절차를 매번 똑같이 반복한다스킬한 번 가르치면 재사용
파일, 터미널, 웹 검색이면 충분하다내장 도구설치 없이 바로 사용
팀에 도구 한 벌을 한 번에 깔고 싶다플러그인묶음으로 배포
비개발자라면

커넥터와 스킬만으로도 대부분 됩니다. MCP는 붙이고 싶은 외부 시스템이 커넥터 목록에 없을 때 꺼내는 카드입니다.

연결하는 법
직접 붙여보기 도구를 붙이는 길은 크게 세 가지입니다

세 가지 연결 방법

🔗

커넥터 추가

클릭으로

Claude 데스크톱이나 웹의 설정에서 커넥터 목록을 열고, 원하는 서비스를 켭니다. 로그인 한 번이면 끝납니다. 코드가 필요 없습니다.

⌨️

claude mcp add

터미널 한 줄

Claude Code에서 한 줄로 외부 MCP 서버를 붙입니다. 커넥터 목록에 없는 서비스도 직접 연결합니다.

🧩

플러그인 설치

묶음으로

스킬과 MCP가 한 벌로 묶인 플러그인을 설치하면, 도구 여러 개가 한 번에 붙습니다. 팀 단위로 도구를 맞출 때 편합니다.

터미널 예시 
# 외부 MCP 서버를 도구로 붙이기
claude mcp add 카카오맵 -- npx -y @playmcp/kakao-map

# 지금 붙어 있는 도구 확인
claude mcp list
실제로 따라해 보기

붙이는 전체 흐름은 실전 예제에서 단계별로 따라갈 수 있습니다. 카카오 도구 연결은 PlayMCP 카카오 예제, 법제처 커넥터 추가는 법제처 MCP 예제를 보세요.
