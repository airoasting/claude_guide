사용 가능

지금까지의 방식을 생각해보세요. 수식이 막히면 → 구글 검색 → 스택오버플로우 → Claude.ai 탭 열기 → 복사 붙여넣기 → 엑셀로 돌아오기. 이 과정을 하루에 몇 번이나 반복하고 있나요?

엑셀 Claude 플러그인은 이 흐름을 끊습니다. 엑셀 우측에 항상 열려 있는 패널에서 작업 중인 셀을 선택한 채로 바로 질문할 수 있습니다. 맥락 전환 없이, 엑셀 안에서 AI 조수와 함께 일하는 방식입니다.

🧮

수식 즉시 생성

막히는 수식을 패널에 설명하면 바로 만들어줍니다. 복사해서 셀에 붙여넣기만 하면 끝.

🔍

오류 즉시 해결

#REF!, #VALUE!, #N/A 오류가 뜨는 셀을 선택하고 "왜 오류가 나?"라고 물으면 원인과 해결책을 알려줍니다.

📊

데이터 분석 제안

범위를 선택하고 "이 데이터로 뭘 볼 수 있어?"라고 물으면 분석 방향과 필요한 수식을 함께 제안합니다.

📚

수식 해설

인수인계받은 복잡한 수식이 이해가 안 될 때, 해당 셀을 선택하고 "이 수식 설명해줘"라고 하면 됩니다.

STEP 2 · 설치하기

5분 설치 완료 가이드

Microsoft AppSource에서 무료로 설치합니다

  1. 엑셀 상단 메뉴 → [삽입] 탭 클릭

    리본 메뉴에서 삽입(Insert) 탭을 선택합니다.

  2. [추가 기능] → [추가 기능 가져오기] 클릭

    또는 상단 리본에서 "Office 추가 기능" 아이콘을 클릭합니다. Microsoft AppSource 창이 열립니다.

  3. 검색창에 "Claude" 입력 후 Anthropic Claude 선택

    Anthropic이 만든 공식 Claude 추가 기능을 선택합니다.

  4. [추가] 버튼 클릭 → 사용 약관 동의

    설치가 완료되면 엑셀 우측에 Claude 사이드패널이 자동으로 열립니다.

  5. Anthropic 계정으로 로그인

    claude.ai에서 사용하는 계정과 동일합니다. Claude 유료 플랜 (Pro / Max / Team / Enterprise) 필수.

💡 설치 후 패널이 안 보이면

상단 리본 메뉴 → [홈] 탭 → 오른쪽 끝의 [추가 기능] 아이콘 클릭 → Claude 선택. 한번 설치하면 이후에는 이 경로로 바로 열 수 있습니다.

⚠️ 회사 PC에서 추가 기능 설치가 막혀 있다면?

많은 기업에서 보안 정책상 AppSource 접근을 차단합니다. 이 경우 IT 부서에 "Anthropic Claude Office Add-in" 허용을 요청하세요. 또는 개인 Microsoft 계정으로 엑셀을 실행하면 우회할 수 있습니다.

STEP 3 · 패널 익히기

사이드패널 첫 화면 이해하기

로그인 후 오른쪽에 열리는 패널 구조입니다

💬

대화 창

PPT 플러그인과 동일한 채팅 인터페이스. 질문을 입력하고 Enter를 누르면 됩니다.

📋

셀 참조 버튼

현재 선택된 셀 또는 범위의 내용을 Claude에게 자동으로 전달합니다. 데이터를 직접 타이핑할 필요가 없습니다.

📌

수식 삽입

Claude가 만든 수식 옆의 [셀에 삽입] 버튼을 누르면 현재 선택한 셀에 자동으로 입력됩니다.

🔄

대화 초기화

새로운 주제로 넘어갈 때는 초기화 버튼으로 대화를 리셋합니다. 이전 맥락이 영향을 주지 않습니다.

📌 핵심 워크플로우

① 셀/범위 선택 → ② 패널에서 질문 → ③ 수식 복사 또는 [삽입] 클릭

이 세 단계가 엑셀 플러그인의 전부입니다. 탭 전환 없이 엑셀 화면을 벗어나지 않고 모든 작업이 완료됩니다.

💡 패널 크기 조절

패널 왼쪽 경계선을 드래그하면 너비를 조절할 수 있습니다. 넓은 모니터라면 패널을 넓게, 노트북이라면 좁게 설정해두면 편합니다.

STEP 4 · 핵심 기능

플러그인으로 할 수 있는 것들

탭 전환 없이 엑셀 안에서 모두 해결합니다

📌 수식 생성 쉬움

데이터 구조를 설명하면 바로 사용 가능한 수식을 만들어줍니다.

A열 날짜, B열 제품명, C열 수량, D열 단가야. E2 셀에 매출액 수식 만들어줘. F2 셀에는 매출 100만원 이상이면 "대형", 50만원 이상이면 "중형", 나머지 "소형" 표시해줘.
=C2*D2
=IF(E2>=1000000,"대형",IF(E2>=500000,"중형","소형"))

📌 오류 해결 쉬움

오류가 난 셀을 선택한 상태에서 질문하면 원인과 수정 방법을 알려줍니다.

E5 셀에 #REF! 오류가 나고 있어. 현재 수식은 =VLOOKUP(B5,시트2!A:C,4,0) 인데 왜 오류가 나는지, 어떻게 고치면 되는지 알려줘.

📌 범위 분석 보통

A1:F50 같은 범위를 선택한 뒤 패널의 [범위 전송] 버튼을 누르고 질문하면, 해당 데이터를 기반으로 분석합니다.

방금 선택한 범위는 1분기 지역별 매출 데이터야. 1. 가장 성과가 좋은 지역 TOP 3를 뽑아줘. 2. 전월 대비 성장률이 가장 높은 제품 카테고리를 알려줘. 3. 이 데이터를 피벗 테이블로 만들려면 어떻게 설정하면 돼?

📌 수식 해설 쉬움

이 수식이 뭘 하는 건지 단계별로 설명해줘: =IFERROR(INDEX(고객!B:B,MATCH(1,(고객!A:A=B2)*(고객!C:C="VIP"),0)),"미등록")
💡 컨텍스트 유지

한 대화 안에서는 이전 내용이 유지됩니다. "방금 그 수식에서 조건을 하나 더 추가해줘"처럼 연속 질문이 가능합니다. 새로운 시트나 주제로 넘어갈 때는 대화를 초기화하세요.

1
뒤죽박죽 데이터, 30초 만에 정리 데이터 정제 · 입문 · 20분

다른 팀에서 받은 고객 명단. 이름 형식 제각각, 전화번호 하이픈 있다 없다, 이메일 대소문자도 뒤섞임. 매달 1시간씩 수작업으로 정리하던 그 비효율. 오늘로 끝냅니다.

프롬프트

[범위] 의 고객 데이터를 정제해줘. - 이름: [원하는 형식] - 전화번호: [원하는 형식] - 이메일: [규칙] - 처리 곤란한 행: [처리 방법] 수정 전에 변경 사항 먼저 보여줘.

체크포인트

  • 변경 사항을 먼저 보여줬는가?
  • 이름·전화·이메일이 원하는 형식으로 통일됐는가?
  • 정제 전후 변경 건수 요약이 나왔는가?
산출물: 깔끔한 고객 DB + 정제 전후 변경 건수 요약
2
수식 무덤에서 탈출 수식 해독 · 기초 · 25분

전임자가 남긴 수식, 고치려면 건드리기도 무섭고 이해도 안 됩니다. 수식 셀을 클릭한 상태에서 질문하면 Claude가 셀 레벨 인용과 함께 즉시 설명합니다.

프롬프트

[셀 주소] 의 수식을 설명해줘. 나는 [직책/배경] 이야. - [내가 가장 궁금한 것] - [또 다른 질문]

체크포인트

  • 각 함수 인수가 무엇을 의미하는지 설명됐는가?
  • 개선된 수식 제안이 포함됐는가?
  • 이해가 안 되는 부분을 후속 질문으로 해결했는가?
산출물: 수식 해설 + 개선된 수식 제안
3
숫자 뒤에 숨은 이야기 찾기 인사이트 도출 · 기초 · 30분

피벗테이블은 완성했는데 이 숫자가 좋은 건지 나쁜 건지 판단이 안 됩니다. 팀장이 "인사이트가 뭐냐"고 물어볼 때 식은땀 나는 그 순간.

프롬프트

[범위] 의 피벗테이블을 [관점: 예) 영업팀장 / CFO] 시각으로 분석해줘. 가장 궁금한 것: [직접 질문 작성] 보고 대상: [누구] 요약: [몇 줄]

체크포인트

  • 요청한 관점(팀장/CFO 등)에서 분석됐는가?
  • 숫자 외에 액션 포인트가 나왔는가?
  • 보고 대상에 맞는 언어와 분량으로 요약됐는가?
산출물: 경영 인사이트 요약 + 액션 포인트
4
보고서 초안, AI로 쓰기 팀 내부 보고서 · 실전 · 20분

매주 금요일 오후 5시. 숫자는 다 나왔는데 글로 쓰는 데 1시간을 씁니다. 팀 내부용 주간 보고. 핵심은 간결함과 다음 주 방향입니다. 결과물은 사이드패널 텍스트로 출력되며, 복사해서 이메일·메신저에 붙여넣어 사용합니다.

프롬프트

[범위] 의 주간 실적으로 [받는 사람] 보고용 초안을 써줘. 형식: [이메일 / 메신저] 분량: [글자 수] 반드시 넣을 것: [직접 결정] 빼야 할 것: [직접 결정]

체크포인트

  • 엑셀 데이터 수치가 그대로 초안에 반영됐는가?
  • 지정한 형식과 분량을 지켰는가?
  • 복사해서 바로 발송 가능한 수준인가?
산출물: 즉시 발송 가능한 팀 내부 보고 초안
5
"만약에..."를 10번 해야 할 때 시나리오 분석 · 실전 · 35분

가격 올리면? 원가가 오르면? 수요가 줄면? Claude가 가정값 셀을 직접 변경하고 수식 의존성을 유지하면서 결과를 보여줍니다. 대화로 연속 시뮬레이션이 가능합니다. 본인의 실제 손익 파일로 진행하세요.

프롬프트

내 손익 모델에서 시나리오 분석을 해줘. 가정값 셀: [직접 찾아서 입력] 시나리오: A. [본인 상황에 맞게 직접 작성] B. [본인 상황에 맞게 직접 작성] 각 시나리오별 [보고 싶은 지표] 를 표로 보여줘. 수정 전에 어떤 셀을 바꿀 건지 먼저 알려줘.

체크포인트

  • 수정 전에 어떤 셀을 바꿀 건지 먼저 알려줬는가?
  • 시나리오별 결과가 표 형태로 나왔는가?
  • 최적안 추천과 근거가 포함됐는가?
산출물: 시나리오 비교표 + 최적안 추천
6
3개 팀 보고서, 한 번에 끝내기 다중 시트 통합 분석 · 심화 · 45분

영업·마케팅·운영 시트를 매달 머릿속에서 연결하는 데 반나절이 걸립니다. Claude는 멀티탭 워크북 전체를 한 번에 읽습니다.

⚠️ 주의

시트당 500행 초과 시 불안정. 요약 시트를 먼저 만들어 전달하고, 날짜 컬럼 형식도 미리 통일해두세요.

프롬프트

[시트명1], [시트명2], [시트명3] 을 통합 분석해줘. 각 시트: [시트명]: [내용 직접 설명] 찾고 싶은 연결고리: - [직접 가설 작성: 예) A팀 성과가 나빠질 때 B팀도 영향 받나?] 인과관계 가설을 [몇 개] 로 정리해줘.

체크포인트

  • 3개 시트 데이터가 모두 반영됐는가?
  • 시트 간 연결고리와 상관관계가 나왔는가?
  • 인과관계 가설이 지정한 개수로 정리됐는가?
산출물: 시트 간 상관관계 분석 + 인과관계 가설
7
임원 보고, 5분 안에 준비 경영진 전략 브리핑 · 마스터 · 40분

CFO가 갑자기 "오늘 오후 3시에 현황 보고해." 경영진은 현황 설명이 아니라 의사결정 근거를 원합니다. 과제 4(팀장 보고)와 구조 자체가 다릅니다: 현황→이슈→옵션→권고. 과제 6 파일을 그대로 활용합니다.

프롬프트

이 파일 전체를 [CFO / CEO] 보고용 브리핑으로 만들어줘. 1. 현황 ([줄 수]): 지금 어디에 있는가, 숫자 근거 포함 2. 이슈 ([개수]): 즉시 결정이 필요한 것 3. 옵션 (각 이슈별 [개수]): 선택지와 트레이드오프 4. 권고안: 네가 [직책] 라면 어떤 선택을 할 건지, 이유 포함 발표 시간 [분] 기준. 초안 말고, 내가 그대로 읽어도 되는 수준으로 써줘.

체크포인트

  • 현황→이슈→옵션→권고 구조가 지켜졌는가?
  • 각 이슈별 선택지와 트레이드오프가 명확한가?
  • 권고안에 이유와 근거가 포함됐는가?
산출물: 경영진 의사결정용 브리핑 + 선택지별 트레이드오프 + 권고안

플러그인 사용 중 생기는 궁금증

Q. 플러그인이 무료인가요?

설치는 무료이나, 사용하려면 Claude 유료 플랜 (Pro / Max / Team / Enterprise)이 필요합니다.

Q. 회사 데이터를 Claude에 보내도 안전한가요?

민감한 데이터(개인정보, 매출 원장 등)는 가명 처리하거나 열 구조와 샘플 5~10행만 전송하세요. 전체 데이터를 보내지 않아도 수식 생성에는 충분합니다. Anthropic의 데이터 처리 정책은 anthropic.com에서 확인할 수 있습니다.

Q. Claude가 만든 수식이 틀릴 수도 있나요?

드물지만 가능합니다. 데이터 구조를 정확하게 설명할수록 정확도가 높아집니다. 항상 샘플 데이터(5~10행)에서 먼저 테스트한 뒤 전체에 적용하세요.

Q. Microsoft 365 Copilot과 다른가요?

다릅니다. Microsoft 365 Copilot은 Microsoft가 만든 AI로 별도 라이선스($30/월)가 필요합니다. Claude 플러그인은 Anthropic이 만든 독립적인 추가 기능으로, 기존 Microsoft 365 구독만 있으면 사용 가능합니다.

Q. Mac용 엑셀에서도 사용할 수 있나요?

네. Mac용 Microsoft Excel(버전 16.0 이상)에서도 동일하게 지원합니다. 삽입 → 추가 기능 경로는 동일합니다.
