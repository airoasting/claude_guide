# 신규 브랜드 기획부터 런칭까지

카테고리나 아이디어 한 마디만 주면, 시장 조사부터 네이밍·아이덴티티·마케팅 플랜까지 8단계로 새 브랜드를 설계한다.

## 무엇을 하나

"간장게장 전문점", "비건 화장품" 같은 출발점 한 줄을 받아 브랜드 전략을 8단계로 쌓아 올린다. 각 단계는 BLACK이 초안을 쓰고 RED·SILVER·BLUE·GOLD 네 페르소나가 차례로 평가해, 9.0점 합격선에 닿을 때까지 다듬는다.

## 언제 쓰나

- 새 브랜드를 처음부터 만들 때
- "브랜드 만들고 싶어", "새 치킨 브랜드 런칭", "프랜차이즈 브랜드 기획" 같은 요청
- 아이디어는 있는데 기획서·가이드·마케팅 플랜으로 한 번에 정리하고 싶을 때

## 입력

카테고리나 아이디어 한 줄. 예: "도시락 구독 앱", "수제 맥주 펍".

## 산출물

- 브랜드 기획서 (STAGE 1~6 통합, 투자자·공동창업자용)
- 브랜드 가이드라인 요약 (1페이지, 디자이너·외주 전달용)
- 마케팅 플랜 (채널·캠페인·KPI, 실행팀용)
- 단계별 5-Color 평가 로그

## 구성

- `SKILL.md` 8단계 프로세스와 5-Color 평가 규칙
- `references/brand-doc.md` 기획서 상세 작성 가이드
- `references/marketing-plan.md` 마케팅 플랜 작성 가이드
- `sample.html` 예시 결과 (치킨 브랜드 데모)

## 이런 건 하지 않는다

- 기존 브랜드 분석·리뉴얼, 네이밍 단독 의뢰
- 광고 카피 한 편 같은 단일 콘텐츠
- 슬라이드·PPT 제작
