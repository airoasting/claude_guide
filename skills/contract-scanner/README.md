# 계약서 스캐너 (Contract Scanner)

계약 문서를 6인 전문가 팀 시각으로 스캔해 위험 조항·누락 조항·협상 포인트를 정리한 결재 보고서(.docx)를 생성하는 Claude 스킬입니다.

## 이 스킬이 하는 일

계약서·약관·합의서·MOU·NDA·공급계약·근로계약 같은 계약 문서를 비법무 담당자가 변호사 검토 전에 1차 스크리닝할 수 있도록 돕습니다. 법무·재무·리스크·협상·도메인·종합 6인 상시 전문가 팀(국제 거래 시 국제법무 1인 추가)이 서로 다른 관점에서 핵심 조항을 추출하고, 독소조항·모호 표현·일방적 조건·누락 조항을 고·중·저 위험도로 표시합니다. 분석 결과는 협상 우선순위 Top 3와 수정 문구를 담은 정식 결재 보고서(.docx)로 자동 정리됩니다. 이 스킬은 법률 자문이 아니라 1차 스크리닝과 결재 보고용 실무 도구입니다.

## 언제 쓰나

다음과 같은 표현을 쓰거나, 계약서·약관으로 보이는 텍스트·PDF·워드·한글(HWP) 파일을 던졌을 때 자동으로 발동합니다.

- "이 계약서 검토해줘", "이 계약 봐줘", "조항 확인해줘"
- "위험 조항 찾아줘", "독소조항 있는지 봐줘"
- "이 약관·NDA·공급계약·MOU 검토해줘"
- "이거 사인해도 되는지", "체결해도 되나"
- "협상 포인트가 뭐야"

다음 경우에는 발동하지 않습니다. 이미 만들어진 계약서의 단순 요약·번역·맞춤법 교정만 원할 때, 계약서를 새로 작성·기안해 달라는 집필 요청일 때, 법률 일반 질문이나 판례 검색만 원할 때입니다.

## 입력과 출력

**입력**

- 계약서 본문 텍스트 직접 입력
- PDF·DOCX·HWP 파일 (텍스트 추출 후 분석)
- 본 계약서 + 별첨(가격표·검수기준표·SLA·정보공개서 등) 다중 파일
- 한·영 혼용 문서 (영문 조항·외국 당사자·외국법 준거 감지 시 국제법무 전문가 자동 추가)

**출력**

기본 산출물은 결재용 워드 문서(.docx)입니다. 다음 순서의 섹션으로 구성됩니다.

1. 표지 ("계약서 검토 보고서")
2. 요약 (최종 등급 + 1줄 결론, 첫 페이지만 봐도 결정 가능)
3. 계약 한눈에 보기 (당사자·목적·기간·금액·지급조건·관할)
4. 위험도 요약 (고·중·저 건수)
5. 조항별 상세 검토 (원문 인용·문제점·협상 제안·검토 의견)
6. 누락된 주요 조항 체크리스트
7. 협상 우선순위 Top 3
8. 종합 결재 의견 (최종 등급·1줄 결론·액션 아이템 3개)
9. 면책 문구 (법률 자문이 아님)

내부 분석에서는 전문가 페르소나와 이모지를 자유롭게 쓰지만, 최종 보고서에는 페르소나 명칭·이모지가 일절 등장하지 않습니다. 모두 "법무 검토 의견", "재무 검토 의견" 같은 정식 직무명으로 변환됩니다. 사용자가 명시적으로 요청하면 PDF·Markdown 형식으로도 출력할 수 있습니다.

## 산출물 예시

보고서가 어떤 모습으로 나오는지는 `sample.html`을 브라우저로 열어 확인할 수 있습니다.

## 사용법

1. `contract-scanner.skill` 파일을 내려받습니다.
2. Claude Desktop에서 Customize > Skill > Add Skill로 이동합니다.
3. 내려받은 `.skill` 파일을 선택해 설치합니다.

설치 후에는 대화에서 자연어로 요청하면 됩니다.

> "거래처에서 받은 계약서입니다. 한번 검토해 주세요."

최종 .docx 보고서 생성에는 `anthropic-skills:docx` 스킬을 함께 사용합니다. 보고서 파일은 사용자가 지정한 경로에 저장되며, 미지정 시 macOS·Linux는 `~/Documents/`, Windows는 `C:\Users\<사용자>\Documents\`에 저장됩니다. 파일명은 `계약검토보고서_<거래처명>_YYYY-MM-DD.docx` 형식을 따릅니다.
