# 매일 투자 브리핑

한국과 미국 시장의 그날 뉴스를 모아 "그래서 투자자에게 무슨 의미인가"까지 정리한 데일리 브리핑을 만든다.

## 무엇을 하나

증시·환율·금리·원자재와 주요 이슈를 웹 검색으로 모으고, 날짜와 출처를 교차 검증한 뒤 사실과 시사점을 함께 묶는다. 강세·약세·리스크 세 관점을 거쳐 한쪽으로 단정하지 않는 브리핑을 낸다.

## 언제 쓰나

- 장 시작 전후로 시장 흐름을 빠르게 잡고 싶을 때
- "오늘 투자 브리핑", "오늘 시장 어때", "데일리 브리핑 줘" 같은 요청
- 코스피·나스닥·환율·금리·관심 종목 동향을 한눈에 보고 싶을 때

## 산출물

채팅 마크다운 브리핑(3~5분 분량), 선택적으로 A4 신문형 PDF. 구성은 다섯 섹션이다.

- 오늘의 헤드라인
- 시장 한눈에 보기 (수치와 등락률)
- 주요 뉴스와 시사점
- 관심 종목과 테마 체크
- 오늘과 내일 체크포인트

## 구성

- `SKILL.md` 7단계 작업 지침과 검증 게이트 (관심 종목은 여기서 바꾼다)
- `scripts/generate_pdf.py` PDF 생성기 (weasyprint, matplotlib)
- `sample.html` 예시 브리핑

## 이런 건 하지 않는다

- 특정 종목 매수·매도 추천
- 개별 기업 재무제표 심층 분석
- 단일 산업만 다루는 브리핑

모든 수치에 출처를 달고 검증된 사실만 본문에 싣는다. 투자 판단의 책임은 본인에게 있다.
