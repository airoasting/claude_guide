# 가정의학과 신규 처방 확대 기획서 초안 (v0)

작성일: 2026-06-23 · 기반: 노바린 인수인계 분석 결과 · 상태: **초안(확정 아님)**

> 이 기획이 "가정의학과 확대"를 1순위로 잡은 이유: 인수인계 분석에서 **가정의학이 브랜드의 핵심 처방군이자 최대 약점**으로 반복 확인됐기 때문. 단, 세부 수치·승인 사항은 아직 가정 위에 있어 6번 항목에서 확인이 필요하다.

## 1. 배경 / 현황
- 노바린 = 고지혈증 치료제, 2026 매출 타겟 **120억(전년比 +18%)** — **[확정]** (출처: 01_브랜드플랜)
- 핵심 처방군: 순환기내과 + 가정의학과 — **[확정]** (출처: 01_브랜드플랜)
- 가정의학과가 약하다 / "Q2 가정의학 드라이브가 연 타겟의 관건" — **[확정]** (출처: 02_월간실적_메모; 1월 "가정의학 약함")
- 따라서 이번 기획의 초점을 가정의학 신규 처방 확대로 둔다 — **[가정/전략 판단]** (위 현황에서 도출)

## 2. 목표
- 가정의학과 신규 처방 매출 비중 확대 — **[가정]** 구체 수치 미정
  - ⚠️ 가정의학 **세부 매출 목표 수치**는 자료에 없음 → 6번에서 상사 확인 필요 (UNKNOWN)
- 연 120억 타겟 달성에 가정의학이 기여하도록 — **[확정 타겟 / 가정 배분]**

## 3. 전략 / 접근
1. 가정의학 핵심 타겟 직접 공략 — 오세영 원장(분당) 신규 미팅 **[확정 타겟, 미접촉]** (출처: 03_KOL_거래처_연락 "신규 처방 확대 타겟, 아직 미팅 못 잡음")
2. KOL 신뢰 자산 활용 — 서준호 교수(4/18 심포 좌장 수락) 연계 레퍼럴 **[확정 관계]** (출처: 03)
3. 디테일 자료 v3 조기 투입 — 단, **광고 심의 통과 확인 후** 배포 **[조건부]** (출처: 04 메모, 06 규제이슈)

## 4. 실행 계획
- [ ] 오세영 원장 미팅 세팅 (미접촉 상태 해소) — 1순위
- [ ] 디테일 v3 광고 심의 대상 여부 확정 → 통과 시 가정의학 디테일링 착수
- [ ] 가정의학 KPI 정의·트래킹 주체 확정 (브랜드 vs 영업)
- [ ] 4분기 예산 재배분안에 "가정의학 비중 근거" 보완해 재상신

## 5. 리스크 / 전제
- **[Blocker]** 약가 인하 대응자료 제출(심평원 5/29, 약가팀 전달 5/22) — **이미 경과, 제출 여부 불명.** 약가 직접 타격 시 매출 전제 자체가 흔들림 (출처: 05 이메일, 06 규제)
- **[Watch]** 디테일 v3 광고 심의 미확인 — 심의 없이 배포 시 컴플 위반 소지 (출처: 06)
- **[전제]** 이 기획은 "가정의학 확대가 옳은 우선순위"라는 판단 위에 섬 → 본부장이 가정의학 비중 근거를 약하다고 반려한 바 있어(출처: 05) **이 전제부터 검증 필요**

## 6. 확인이 필요한 빈칸 (Open Questions) — 이 초안이 서 있는 가정들
1. 가정의학 **세부 매출 목표**는 얼마인가? (자료에 없음 → 상사) 
2. 본부장이 요구한 **"가정의학 비중 근거"** 의 구체 기준은? (출처: 05, 반려 사유)
3. 가정의학 **KPI 트래킹 주체**가 브랜드인가 영업인가? (출처: 04 "누가 트래킹? 나? 영업팀?")
4. 약가 대응자료가 **실제 제출됐는가?** (제출 안 됐으면 이 기획보다 약가 대응이 선행)

> 이 초안은 4개의 가정 위에 서 있다. 위 4개가 확정되기 전까지는 방향 제안일 뿐 확정 계획이 아니다.
