# 예시 산출물 (examples)

`sample/` 더미(가상의 퇴사자 자료)를 Handover Navigator로 실제 처리해 나온 결과물입니다. **스킬을 설치하지 않아도** 이 폴더만 보면 산출물이 어떤 모습인지 미리 확인할 수 있습니다. 모든 데이터는 가상(노바린/한빛제약)입니다.

| 파일 | 산출물 |
| --- | --- |
| `handover-summary-EXAMPLE.md` | 인수인계 요약서 (1페이지 + 6블록 A~G) |
| `open-tasks-tracker-EXAMPLE.md` | 미결 과제 트래커 (14건, 상태·담당·기한·신뢰도·출처) |
| `onboarding-30day-EXAMPLE.md` | 첫 30일 온보딩 플랜 |
| `questions-to-ask-EXAMPLE.md` | 물어볼 질문 리스트 (15문항, 3단계 우선순위) |
| `new-plan-draft-EXAMPLE.md` | 신규 기획서 초안 (확정/가정 구분) |

## 이 예시들이 보여주는 핵심

- **모든 사실에 (출처: 파일) 표기** — 근거 없는 단정이 없음
- **상태 정규화** — "거의 끝"→진행중, "홀딩"→막힘, "?"→불명
- **날짜 경과 포착** — 5/22·5/29 마감을 작성일(6/23)과 비교해 경과+Blocker로
- **모르는 것은 UNKNOWN** → "물어볼 질문 리스트"로 이관 (지어내지 않음)

스킬 없이 같은 더미를 요약했을 때와의 대조는 `../evals/eval-report.md`(RED vs GREEN) 참조.
