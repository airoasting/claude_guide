# 예비 가맹점주 적격성 진단

본사 담당자가 예비 가맹점주 상담 자료를 받아, 이 후보를 진행할지 보류할지 거절할지 근거와 함께 판단하도록 돕는다.

## 무엇을 하나

상담 메모나 설문 응답을 다섯 페르소나가 각자 시각으로 본다. 자금·운영·상권·브랜드 네 영역을 10점으로 채점하고, 정보가 부족한 영역은 추측 대신 N/A로 둔다. 거부권 여부를 따로 표기해 최종 판정과 8점 도달 로드맵까지 낸다.

## 다섯 페르소나

| 페르소나 | 역할 | 거부권 |
|---|---|---|
| 블랙 | 총괄, 종합점수와 최종 합의 | |
| 레드 | 리스크, 자금·부채·가족 동의 | 있음 |
| 골드 | 현장 운영, 경험·근무 의지·체력 | 있음 |
| 블루 | 상권, 입지·유동인구·경쟁 | |
| 화이트 | 브랜드, 협업 태도·표준 준수 | |

## 언제 쓰나

- 예비 가맹점주 상담 메모나 전화·미팅 노트를 분석할 때
- "이 점주 후보 적격성 평가해줘", "진행할지 보류할지 판단해줘" 같은 요청
- 같은 상담자의 2차 자료가 오면 이전 점수와 자동으로 비교한다

## 입력

상담 메모, 통화 노트, 또는 설문 응답. 모바일 설문 앱(`app/questionnaire_app.html`)으로 받아 붙여넣어도 된다.

## 산출물

페르소나별 채점과 근거, 종합 판정(적합·보류-정보부족·보류-개선조건부·부적합), 8점 도달 로드맵, 다음 상담에서 물어볼 질문.

## 구성

- `SKILL.md` 채점·거부권·판정 규칙
- `docs/DECISION_RULES.md` 판정 규칙 정본 (본문과 충돌 시 우선)
- `docs/questionnaire.md`, `app/questionnaire_app.html` 설문 양식과 입력 앱
- `poc/` 검증 케이스 네 건

## 이런 건 하지 않는다

- 자료에 없는 정보를 추측해 채점
- 계약 유도를 위한 점수 부풀리기
- 법률·세무·대출 가능 여부 단정
- 가맹 본사 매뉴얼 작성 (franchise-manual 영역)
