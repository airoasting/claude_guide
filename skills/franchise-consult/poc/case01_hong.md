# PoC Case 01 — 홍길동 (1차 상담)

## 입력 데이터 (요약)
- 40세, 세종시 거주, 1남 2녀
- 자기자본 1억(즉시현금 3천), 대출 계획 없음, 6개월 예비자금 없음
- 외식업·직원관리 무경험, 부업 병행, 자신 업무: 조리 / 부담: 고객응대
- 후보 점포 없음, 상권 인식 없음, 입지 미정
- 브랜드 인지 경로: 유튜브 / 선택 이유: "매출과 이익이 좋다"
- 가족 미상담, 실패 대비책 없음

## 페르소나별 점수
| 페르소나 | 점수 |
|---|---|
| 레드 (리스크) | 2.5 / 10 |
| 골드 (운영) | 3.0 / 10 |
| 블루 (상권) | N/A |
| 화이트 (브랜드) | 5.5 / 10 |
| **종합** | **3.7 / 10** (N/A 제외) |

## 거부권
- **레드**: 자금 결정적 부족 + 가족 미상담 + 실패 대비책 없음
- **골드**: 무경험 + 부업 병행 + 고객응대 부담

## 최종 판정
**부적합**

## 8점 도달 로드맵 (요약)
- [레드 2.5 → 8.0] 즉시 가용 현금 6천 + 예비자금 2천 + 가족 동의 + 회수 인식 24~36개월
- [골드 3.0 → 8.0] 부업 → 전업 전환 + 직영점 1개월 체험 + 고객응대 보완 인력
- [블루 N/A → 8.0] 희망 상권 명시 + 직접 답사 + 경쟁 매장 3곳 인지 + 임대료 예산
- [화이트 5.5 → 8.0] 본사 자료 학습 + 직영점 방문 + 경쟁 브랜드 2곳 비교

## 검증된 시스템 기능
- 거부권 동시 발동 처리 (레드 + 골드)
- 블루 영역 N/A 처리 (정보 절반 이하 → 추측 채점 차단)
- 페르소나별 우선순위 질문 자동 도출 (총 6개)
- 8개 정보 공백 항목을 우선순위와 함께 정리

## 시사점
"관심은 강하지만 준비가 전혀 안 된" 전형적 패턴. 본사 입장에서 단순 거절보다 8점 도달 로드맵을 안내하며 재상담 기회를 제공하는 것이 평판 리스크와 점주 풀 관리 양면에서 유리.
