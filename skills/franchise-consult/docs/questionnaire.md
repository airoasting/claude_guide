## 사용 방법
1. 위 항목 중 답할 수 있는 것만 채워서 클로드에게 그대로 붙여넣습니다.
2. **빈 항목은 "미정"·"모름"·"미상담"** 중 적절한 표시를 남깁니다 (지우지 마세요. 어떤 항목이 비었는지 자체가 분석 신호입니다).
3. 클로드가 자동으로 5인 페르소나 분석 → 블랙의 최종 판단(적합/보류-정보부족/보류-개선조건부/부적합)을 출력합니다.
4. 같은 상담자의 2차 상담 결과를 다시 붙여넣으면 이전 평가와의 변화·잔존 리스크가 비교 출력됩니다.
