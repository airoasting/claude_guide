# Usage Guide — 처음 사용자용 안내

## 1. 이 스킬이 하는 일

예비 가맹점주와의 상담 메모를 받아 자동으로 다음을 출력합니다.

- 자금·운영·상권·브랜드 4개 영역의 10점 만점 채점 + 평균 종합점수
- 거부권 발동 여부 (자금 결정적 부족, 운영 의지 부재 등)
- 최종 가맹 판단: **적합 / 보류-정보부족 / 보류-개선조건부 / 부적합**
- 8점 도달 개선 로드맵 (점주 육성 가이드)
- 다음 통화에서 확인할 질문 리스트

## 2. 두 가지 사용 방법

### 방법 A — 모바일 앱 사용 (권장)

1. `app/questionnaire_app.html`을 브라우저에서 엽니다. 모바일에서도 잘 작동합니다.
2. "+ 새 상담 시작" 버튼으로 폼 진입.
3. A~F 6개 섹션을 채웁니다. 모르는 항목은 비워두세요(추측 입력 금지).
4. 자동 저장됩니다(30초마다). 화면 상단에서 입력 완성도가 실시간 표시됩니다.
5. 하단 "분석용 출력" 버튼 → 정형 마크다운 자동 생성 → "복사" 버튼
6. Claude 채팅에 붙여넣기 → 자동 분석 시작

### 방법 B — 텍스트 직접 입력

상담 메모를 자유 서술로 작성하거나, `docs/questionnaire.md`의 양식을 복사해 채운 뒤 Claude 채팅에 붙여넣어도 됩니다. 다만 자유 서술은 누락 정보가 많아 평가 정확도가 떨어질 수 있어 앱 사용을 권장합니다.

## 3. 출력 해석하기

### 종합점수
4개 페르소나(레드·골드·블루·화이트) 점수의 단순 평균입니다. 8점 이상이면 적합 구간이지만, **거부권 발동 시 점수와 무관하게 부적합**으로 결정됩니다.

### 거부권
레드(리스크)와 골드(운영)는 각자 거부권을 가집니다. 자금이 결정적으로 부족하거나 운영 의지·경험이 결정적으로 부재한 경우 점수와 별개로 부적합 판정이 내려집니다.

### 보류 2단계
- **보류-정보부족**: 결정에 필요한 핵심 정보가 누락되어 판단 불가. 추가 정보 확보가 우선.
- **보류-개선조건부**: 정보는 충분하나 특정 조건(자기자본 추가 확보, 본업 정리 등)이 충족되면 적합 전환 가능.

### N/A 처리
어떤 영역의 정보가 절반 이하로만 들어왔으면 그 영역은 추측 채점하지 않고 N/A로 표기합니다. 종합점수는 N/A 영역을 제외한 평균으로 산출됩니다.

### 8점 도달 로드맵
부적합·보류 판정 케이스에도 "현재 X.X점 → 8점이 되려면 무엇이 필요한가"를 페르소나별로 명시합니다. 본사 영업기획팀이 상담자에게 구체적 개선 안내를 할 때 그대로 활용 가능합니다.

## 4. 2차 상담 분석 (재분석 모드)

같은 상담자의 추가 메모를 다시 붙여넣으면 자동으로 재분석 모드에 진입합니다.
- 영역별 점수 변화 (▲ / ▼ / -)
- 거부권 변화
- 개선된 항목 / 잔존 리스크 / 신규 리스크
- 판정 변화

모바일 앱에서는 같은 레코드를 불러와 보완 후 다시 출력하면 됩니다.

## 5. 주의사항

- 상담자에게 점수를 그대로 공개할지는 본사 판단입니다. 점수는 내부 의사결정 도구이지 협상 도구가 아닙니다.
- 법률·세무·대출 가능 여부에 대해 본사가 단정적으로 답하지 마세요. 외부 전문가 연결만 안내합니다.
- 거부권 발동 케이스를 "그래도 가능성 있음"으로 흐리게 결론짓지 마세요. 8점 도달 로드맵으로 명확히 안내하는 것이 본사 평판 리스크를 줄이는 길입니다.

## 6. PoC 케이스 살펴보기

`poc/case01_hong.md`와 `poc/case02_id123.md`에서 실제 분석 출력 예시를 확인할 수 있습니다.

- 케이스 1: 무경험 + 자금 부족 + 부업 병행 → 종합 3.7점, 레드·골드 동시 거부권, **부적합**
