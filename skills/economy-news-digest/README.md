# 경제 뉴스 다이제스트

요청한 그 시점을 기준으로 한국 경제 뉴스를 검색해 증시·금리·환율·부동산 네 카테고리로 정리하는 데일리 브리핑 스킬입니다.

## 이 스킬이 하는 일

요청을 받으면 그 시점에 웹을 검색해 최신 한국 경제 뉴스를 수집하고, 증시·금리·환율·부동산 네 카테고리로 나눕니다. 각 카테고리는 뉴스 앵커 톤의 담백한 사실 요약(2~3줄)으로 정리합니다. 여기에 더해, 투자 철학이 서로 다른 두 전문가 아키타입이 같은 사실을 각자의 분석 틀로 해석하고 앞으로의 전망을 제시합니다. 같은 지표 하나로도 보는 틀에 따라 다른 결론이 나올 수 있다는 점을, 독자가 스스로 판단하도록 입체적으로 보여주는 것이 목표입니다.

## 언제 쓰나

- "경제 뉴스 요약해줘", "오늘 경제 소식 알려줘"
- "오늘의 증시·금리·환율·부동산 정리해줘"
- "아침 경제 브리핑", "경제 동향 정리"
- "경제 다이제스트", "/economy-news-digest"

다음 경우에는 발동하지 않습니다. 특정 종목 매수·매도 분석, 개인 맞춤 투자 조언, 과거 특정 날짜의 뉴스 아카이브 조회, 경제와 무관한 일반 뉴스 요약입니다.

## 입력과 출력

자연어로 "오늘 경제 뉴스 요약해줘" 수준의 요청을 주면, 네 카테고리 브리핑이 한 묶음으로 나옵니다.

- 카테고리 순서는 증시 → 금리 → 환율 → 부동산으로 고정됩니다.
- 각 카테고리는 사실 요약(2~3줄) 다음에 두 전문가 코멘트가 이어집니다.
- 각 코멘트는 "해석(현재를 어떻게 보는가)"과 "전망(앞으로 어떻게 될 것으로 보는가)"을 한 줄씩 제시합니다.
- 날짜는 검색을 수행한 실제 현재 날짜로 적고, 마지막 줄에 검색 시점 기준임을 한 줄로 덧붙입니다.

전문가는 실존 인물이 아니라 유명 투자 철학·학파를 대표하는 익명 아키타입입니다. 실명이나 그들의 발언 인용은 쓰지 않습니다.

| 카테고리 | 철학 A | 철학 B |
|---|---|---|
| 증시 | 가치투자 (내재가치·안전마진) | 성장투자 (확장성·혁신 곡선) |
| 금리 | 통화주의 (물가·기대인플레) | 케인스주의 (총수요·고용) |
| 환율 | 펀더멘털 (금리차·경상수지) | 플로우·심리 (자금·위험선호) |
| 부동산 | 입지·수급 | 유동성·거시 |

출력 골격은 다음과 같습니다.

```
📊 오늘의 경제 다이제스트 (YYYY년 M월 D일)

📈 증시
(2~3줄 사실 요약)
🗣️ 가치투자 철학
  해석: (1줄)
  전망: (1줄)
🗣️ 성장투자 철학
  해석: (1줄)
  전망: (1줄)

(금리·환율·부동산도 동일한 구조로 이어집니다)

※ M월 D일 H시경 검색 기준이며, 이후 상황은 달라질 수 있습니다.
```

## 사용법

Claude에 스킬로 등록한 뒤 자연어로 요청하면 됩니다.

1. `SKILL.md`를 Claude 스킬 폴더에 넣습니다.
2. Claude에게 "오늘 경제 뉴스 요약해줘"라고 요청합니다.

이 스킬은 요청 시점에 웹을 검색해 동작하므로 별도의 API 키나 환경변수, MCP 연결이 필요하지 않습니다. 다만 "매일 오전 정해진 시각 자동 실행" 같은 스케줄 기능은 스킬 자체에 없습니다. 자동화하려면 cron 같은 외부 스케줄러와 연결하면 됩니다. 이 스킬은 "요약하는 두뇌" 역할을 담당합니다.
