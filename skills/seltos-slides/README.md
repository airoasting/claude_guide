# seltos-slides · 트림·옵션 구매 의사결정 슬라이드

트림과 옵션이 여러 단계로 갈라지는 고관여 상품을 놓고 "결국 뭘 사야 하는가"에 답하는 비교·추천 산출물을 만드는 스킬입니다.

## 이 스킬이 하는 일

자동차처럼 트림과 옵션이 복잡하게 얽힌 상품의 구매 의사결정을, 서로 다른 세 시선(판매 컨설턴트, 실사용 구매자, 시각 디자이너)으로 따로 평가해 비교합니다. 정가표를 그대로 옮기는 일이 아니라, 같은 후보 트림을 세 페르소나가 각자 채점하고 그 차이를 드러내 추천이 한쪽으로 치우치지 않게 합니다. 결과는 발표·공유용 슬라이드덱 또는 한 페이지로 죽 읽는 단일 스크롤 리포트로 나옵니다.

## 언제 쓰나

- "셀토스 구매 비교", "트림 추천 슬라이드 만들어줘" 같이 트림·옵션 비교를 요청할 때
- "어떤 트림 사야 해", "이 옵션 넣을 가치 있나" 같은 구매 판단을 원할 때
- "옵션 가성비 비교", "트림별 견적 정리" 처럼 구성 조합이 가격을 크게 흔드는 상품을 정리할 때
- 자동차뿐 아니라 캠핑카, 가전 풀패키지, 커스텀 PC·노트북 등 트림·패키지가 여러 단계로 나뉜 고관여 상품의 비교·추천

다음 경우에는 발동하지 않습니다.

- 제원·스펙 단순 조회("셀토스 전장 얼마야")
- 한 트림만 설명하거나 정가표를 그대로 정리("프레스티지 가격만 알려줘")
- 중고 시세·실매물 조회, 보험료 견적 같은 외부 실시간 데이터가 본질인 요청
- 계약서·할부 계산이 핵심인 요청

## 입력과 출력

다음을 입력으로 받습니다. 비어 있으면 합리적 기본값으로 채우거나 "(미입력)"으로 표기합니다.

| 입력 | 설명 |
|---|---|
| 대상 차종·모델, 연식 | 비교 대상 |
| 비교할 트림 범위 | 전체인지, 중상위만인지 |
| 구매자 우선순위 | 가성비·안전·디자인·연비 중 무엇을 더 보는지 (종합점수 가중치) |
| 파워트레인 관심 | 가솔린·터보·하이브리드 등 |

출력은 다음 순서로 구성됩니다(단일 스크롤 리포트 기준).

1. 헤더: 모델명, 연식, 부제, 메타 배지(비교 트림 수, 파워트레인, 기준일)
2. 입력 요약: 이번 비교의 전제 한 줄
3. 트림 비교표: 트림 x 파워트레인 가격 매트릭스, 핵심 옵션 포함 여부
4. 페르소나 평가: 세 명 x 후보 트림 별점·코멘트 카드
5. 가중 종합: 축별 점수와 가중치, 종합 순위
6. 최종 추천: 1순위 트림 + 근거 + 조건부 대안
7. 면책: 가격·제원은 변동될 수 있으며 실제 견적은 대리점 확인 필요

슬라이드덱이면 같은 순서를 1280x720 슬라이드 3~5장으로 나눕니다. 가격·연비·제원은 모르면 지어내지 않고 "확인 필요" 또는 "추정"으로 라벨링하며, 페르소나는 실존 인물이 아닌 가상의 역할입니다.

## 산출물 예시

실제 출력 형식은 이 폴더의 `sample.html`을 열어 확인하세요. 2026 기아 셀토스 트림·옵션 비교를 단일 스크롤 리포트로 한 번 돌린 결과물로, 트림 비교표·세 페르소나 별점·가중 종합·최종 추천 트림까지 담겨 있습니다.

## 사용법

Claude에 스킬로 등록한 뒤 자연어로 요청하면 됩니다. 예시.

```
2026 셀토스 프레스티지부터 시그니처까지 비교해서 추천 리포트 만들어줘. 가성비랑 안전 우선으로.
```

```
캠핑카 트림별 옵션 가성비 비교 슬라이드 만들어줘
```

형태 언급이 없으면 비교 항목이 많고 근거를 길게 남겨야 할 때는 리포트를, 발표·공유 뉘앙스가 있으면 슬라이드를 고릅니다. 애매하면 한 번 되묻습니다.
