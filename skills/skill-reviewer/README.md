# 스킬 문서 채점·개선

SKILL.md나 CLAUDE.md 같은 스킬 문서를 네 기준으로 채점하고, 개선본을 새 파일로 만든다.

## 무엇을 하나

대상 문서를 끝까지 읽고 목적·오케스트레이션·역할 배정·토큰 효율 네 항목을 10점으로 채점한다. 평균 점수와 근거를 리포트로 내고, 개선본을 원본 옆에 새 파일로 저장한다. 원본은 절대 고치지 않는다.

## 언제 쓰나

- "내 SKILL.md 검토해줘", "이 CLAUDE.md 10점으로 점수 매겨줘" 같은 요청
- 스킬 문서의 품질을 감사하고 싶을 때

## 입력

검토할 파일 경로. 지정하지 않으면 `**/SKILL.md`, `./CLAUDE.md` 등을 찾아 읽는다.

## 산출물

- 네 항목 점수와 평균이 담긴 채점 리포트
- 개선본 파일 `*.improved.md` (이름이 겹치면 `.improved.2.md` 식으로 늘린다)

## 구성

- `SKILL.md` 4단계 채점 워크플로우와 감점 신호
- `sample.html` 예시 리포트

## 이런 건 하지 않는다

- 원본 파일 수정
- 새 스킬 작성 (skill-creator 영역)
- 일반 마크다운 교정이나 오타 수정
