# Chart-Kakao Analyst (차트 카카오 애널리스트)

주식 차트 이미지를 혼마 무네히사 캔들 분석법으로 진단하고, 결과 브리핑을 카카오톡 나에게 보내기로 전송하는 스킬입니다.

## 이 스킬이 하는 일

차트 스크린샷을 읽어 캔들 패턴, 거래량, 이동평균선 배열, 보조지표를 순서대로 해석합니다. 가격 행동(캔들)에서 신뢰도(거래량), 추세(이동평균), 과열도(보조지표) 순으로 검증하며, 마지막에 매수·관망·매도 신호 하나로 종합 판단합니다. 분석 결과는 카카오톡 나에게 보내기로 바로 전송되며, 카카오 연결이 없으면 대화창에 그대로 출력합니다. 차트 이미지의 기술적 패턴 해석과 전송만 담당하며, 재무제표 분석이나 시세 조회·매매 주문은 다루지 않습니다.

## 언제 쓰나

다음과 같이 요청하면 발동합니다.

- "차트 분석해줘", "이 차트 봐줘"
- "Chart 폴더 분석", "오늘 차트 봐줘"
- "혼마 분석", "캔들 분석", "기술적 분석 해줘"
- "카카오로 차트 보내줘", "차트 카카오톡으로"
- 차트 이미지를 첨부하며 기술적 진단을 요청할 때

다음 경우에는 발동하지 않습니다. 기업 재무제표·실적 분석(DART 리포트 영역), 뉴스·공시 텍스트 요약, 단순 시세 조회, 매매 주문 실행 요청.

## 입력과 출력

입력은 주식 차트 이미지입니다. 두 가지 방식을 지원합니다.

- 대화에 차트 이미지를 직접 첨부하는 방식
- 지정한 Chart 폴더에 이미지를 모아두는 방식 (지원 확장자: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.bmp`)

출력은 종목별 기술적 분석 브리핑입니다. 카카오톡 텍스트로 그대로 보낼 수 있는 형식이며, 다음 블록으로 구성됩니다.

| 블록 | 내용 |
|------|------|
| 혼마 무네히사 캔들 진단 | 적삼병·흑삼병·도지·망치형·샛별/저녁별형 등 발견된 패턴과 의미 |
| 거래량 해석 | 거래량 추이와 가격 신뢰도 평가 |
| 이동평균선 위치 | 5/20/60/120일선 배열, 골든/데드크로스, 지지·저항 구간 |
| 보조지표 | RSI 과매수·과매도, MACD 시그널 교차 (차트에 보일 때만) |
| 종합 판단 | 매수·관망·매도 신호등 하나 + 한줄 요약 + 상세 의견 |

종합 판단은 신호등 셋 중 정확히 하나만 고르며, 애매하면 관망을 기본값으로 둡니다. 차트에 없는 가격·날짜·지표 수치는 추정으로 채우지 않으며, 모든 브리핑에 투자 면책 문구를 포함합니다.

## 사용법

이 스킬은 두 가지 운영 경로를 지원합니다.

### Claude 스킬로 사용

Claude에 스킬로 등록한 뒤 자연어로 요청하면 됩니다. 차트 이미지를 첨부하거나 Chart 폴더에 모아둔 상태에서 "차트 분석해줘"라고 말하면 혼마 분석 후 카카오톡(`KakaotalkChat-MemoChat`, 카카오 MCP의 나에게 보내기)으로 전송합니다. 카카오 MCP를 쓸 수 없으면 결과를 대화창에 출력합니다.

### 폴더 감시 자동화로 사용 (watcher.py)

Chart 폴더에 이미지를 저장하면 자동으로 분석·전송하는 Python 스크립트가 함께 들어 있습니다.

설치

```bash
pip install -r requirements.txt
```

환경 변수 설정. `.env.example`을 `.env`로 복사하고 키를 입력합니다.

```bash
cp .env.example .env
```

- Anthropic API 키: [console.anthropic.com](https://console.anthropic.com) → API Keys
- 카카오 액세스 토큰
  1. [Kakao Developers](https://developers.kakao.com) → 내 애플리케이션 → 앱 추가
  2. 앱 설정 → 카카오 로그인 → 활성화
  3. 동의항목 → `카카오톡 메시지 전송` 동의 설정
  4. 도구 → REST API 테스트 → 토큰 발급 (나에게 보내기 권한: `talk_message`)
  5. 발급된 액세스 토큰을 `.env`의 `KAKAO_ACCESS_TOKEN`에 입력

액세스 토큰 유효기간은 6시간입니다. 장기 운영 시 리프레시 토큰으로 갱신하거나 Kakao OAuth 자동 갱신 로직을 추가하세요.

실행

```bash
python watcher.py
```

실행 후 `Chart/` 폴더에 차트 이미지를 저장하면 자동으로 분석이 시작됩니다. `KAKAO_ACCESS_TOKEN`을 비워두면 분석 결과를 콘솔에 출력합니다. API 키와 액세스 토큰은 GitHub에 커밋하지 마세요(`.gitignore`에 포함되어 있습니다).

```

├── .env.example        # 환경 변수 템플릿
├── .gitignore
├── sample.html         # 브리핑 산출물 예시
└── LICENSE
```
