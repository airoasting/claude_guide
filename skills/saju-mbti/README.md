# 사주·MBTI 통합 상담

명리학과 성격검사를 결합해, 다섯 전문가가 보고 한 명의 상담사가 통합해 답하는 인생 상담을 제공한다.

## 무엇을 하나

사주팔자와 오행, MBTI와 TCI, 그리고 사진이 있으면 관상과 손금까지 함께 본다. 다섯 페르소나가 각자 근거로 소견을 내고, 종합상담사가 고민의 성격에 따라 비중을 조절해 하나의 답으로 묶는다. 사주는 항상 만세력 스크립트로 계산하며 기억으로 지어내지 않는다.

## 언제 쓰나

- "나에 대해 알려줘", "내 사주 봐줘", "성격검사 해줘", "조언이 필요해" 같은 요청
- 진로·적성·인간관계·연애·작명 같은 고민을 다각도로 보고 싶을 때

## 입력

- 양력 생년월일 (필수), 출생 시각과 출생지 (선택)
- MBTI나 TCI 결과 (없으면 간이검사 제공)
- 얼굴·손바닥 사진 (선택, 없으면 관상·손금 제외)

대화로 부담 없이 나눠서 받는다.

## 산출물

고민 요약과 사주 한눈에 보기, 다섯 전문가 소견, 종합상담사의 답과 실천 두세 가지, 마무리. 이후 대화는 상담 메모로 맥락을 이어간다.

## 구성

- `SKILL.md` 6단계 워크플로우와 가중 규칙, 가드레일
- `scripts/saju_calculator.py` 만세력 계산 엔진
- `references/` 명리·성격검사·관상·손금 해석 자료
- `sample.html` 예시 상담

## 이런 건 하지 않는다

- 단정이나 예언 (운명·수명·질병)
- 음력을 양력처럼 계산
- 타로·별자리 같은 다른 점술

위기 신호가 보이면 점술 틀을 내려놓고 도움받을 곳을 안내한다.
