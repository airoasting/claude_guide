# 플랫폼별 검색 전략 & 데이터 접근 현실

세 플랫폼은 데이터 접근성이 크게 다르다. 각 플랫폼의 현실을 알고, 그에 맞춰 검색하고,
출처 신뢰도를 정직하게 태깅한다. 핵심: **접근이 어려운 플랫폼일수록 간접 출처가 많아지고,
간접 출처는 반드시 `[간접]`으로 표기**한다.

---

## 1. X (트위터) — 접근성 ★★★ (가장 잘 됨)

실제 게시물이 웹 검색에 잘 잡힌다. 직접 인용을 가장 많이 확보할 수 있는 플랫폼.

**검색 쿼리 패턴**
- `[키워드] X` / `[키워드] 트위터 반응`
- `[keyword] twitter reaction` / `[keyword] x.com`
- 한국 이슈면 한국어, 글로벌 이슈면 영어, 필요 시 일본어/중국어도.

**수집 방법**
- 검색 결과에 `x.com/.../status/...` 링크가 있으면 web_fetch로 원문·반응 확인.
- 인용수/좋아요 많은 게시물, 그리고 그 아래 리플라이 여론을 함께 본다.
- 한 게시물에서 인용은 1개, 15단어 이내로 짧게. 번역 병기.

**주의**
- 비공개/삭제 게시물은 못 가져온다. 검색 시점의 공개 글만 사용.
- 봇·어뷰징성 반복 글은 여론으로 과대 해석하지 않는다.

---

## 2. 샤오홍슈 (小红书 / RED / xiaohongshu) — 접근성 ★★☆ (부분적으로 됨)

뷰티·패션·여행·맛집·육아 등 라이프스타일 후기가 강한 플랫폼. 한국 관련 콘텐츠도 많다.
일부 노트(게시물)는 웹 검색에 잡히고, 한국 블로그·뉴스가 RED 트렌드를 정리한 글도 많다.

**검색 쿼리 패턴**
- `[키워드] 小红书` / `[키워드] 샤오홍슈 후기` / `[키워드] 샤오홍슈 추천`
- `xiaohongshu [keyword]` / `[키워드] 小红书 测评`(리뷰) / `[키워드] 小红书 种草`(추천)
- 한국 관련: `[키워드] 샤오홍슈 한국`, `[키워드] 小红书 韩国`

**수집 방법**
- `xiaohongshu.com/...` 또는 `xhslink.com/...` 링크가 보이면 web_fetch 시도(직접 출처).
- 직접 노트를 못 찾으면, RED 트렌드를 정리한 한국/중국 블로그·뉴스를 간접 출처로 사용.
- 인용은 원문(중국어) + 한국어 번역. 출처 태그 직접/간접 명확히.

**주의**
- RED는 "种草(제품 추천)" 특성상 광고성·협찬 글이 섞인다. 자연 후기와 구분이 어려우면
  "광고성 가능성 있음"을 메모한다.
- 검색이 막히면 키워드를 중국어 简体로 바꿔 다시 시도.

---

## 3. 더우인 (抖音 / 중국판 틱톡) — 접근성 ★☆☆ (가장 어려움, 대부분 간접)

영상 중심 + 만리방화벽 안에 있어 **실제 게시물을 웹으로 직접 보기는 거의 불가능**하다.
따라서 더우인 분석은 **간접 출처 중심**이 되며, 이를 숨기지 말고 그대로 표기한다.

**현실 직시**
- `douyin.com` 원문이 검색에 잡히는 경우는 드물다. 잡히면 `[직접]`으로 시도.
- 대부분은 아래 간접 경로로 트렌드의 "존재와 방향"을 확인한다.

**간접 출처로 쓸 수 있는 것 (모두 `[간접]`)**
- 더우인 인기/핫서치 리포트를 인용한 중국 마케팅·뉴스 매체 글 (예: 抖音热点, 巨量算数 언급 기사)
- 한국어로 더우인 트렌드를 정리한 블로그·뉴스·마케팅 글
- 더우인에서 화제가 된 콘텐츠가 X/샤오홍슈/유튜브 등으로 재유통된 흔적

**검색 쿼리 패턴**
- `[키워드] 抖音` / `[키워드] 抖音 热点` / `[키워드] 抖音 热搜`
- `[키워드] 더우인 트렌드` / `[키워드] 더우인 인기` / `douyin [keyword] trend`
- `[키워드] 抖音 挑战`(챌린지) / `[키워드] 抖音 话题`(해시태그 주제)

**주의 (가장 중요)**
- 더우인 섹션에서는 "실제 댓글을 봤다"는 식의 인상을 주지 않는다.
  확인한 것은 대개 "트렌드의 방향/규모를 2차 자료로 추정"한 수준임을 분명히 한다.
- 더우인 데이터를 전혀 못 찾으면 "더우인: 이번 주제 관련 확보 데이터 없음"이라고 솔직히 적는다.

---

## 출처 신뢰도 우선순위 (높음 → 낮음)
1. 실제 공개 게시물 원문 (직접)
2. 플랫폼 공식/준공식 트렌드 지표를 인용한 매체 (간접, 신뢰 높음)
3. 평판 있는 뉴스·산업 매체의 트렌드 정리 (간접)
4. 개인 블로그·마케팅 글의 2차 정리 (간접, 교차 확인 필요)
5. 출처 불명 / 단일 미확인 주장 → 사용하지 않는다

여러 플랫폼·여러 출처에서 같은 신호가 보이면 신뢰도를 함께 언급한다
(예: "X 직접 글 + 샤오홍슈 후기 + 더우인 트렌드 리포트에서 공통 확인 → 신뢰도 높음").
