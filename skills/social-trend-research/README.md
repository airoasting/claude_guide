# SNS 트렌드·여론 리서치 (더우인 · 샤오홍슈 · X)

더우인(抖音, 중국판 틱톡)·샤오홍슈(小红书/RED)·X(트위터)의 실제 공개 게시물을 근거로 여론·트렌드를 한국어 리포트로 만드는 Claude 스킬입니다.

## 이 스킬이 하는 일

학습 데이터의 기억이 아니라 지금 사람들이 실제로 하는 말을 웹에서 찾아 분석합니다. 특정 제품·브랜드·인물·이슈·키워드에 대한 공개 게시물을 검색·확인하고, 실제 인용과 출처를 단 구조화된 리포트로 정리합니다. 핵심 가치는 "실제로 찾은 것만 쓴다, 환각 없음"이며, 검색으로 확인하지 못한 인용·수치·게시물은 만들어내지 않고 비워두고 솔직히 적습니다.

## 언제 쓰나

다음과 같은 요청이 오면 발동합니다.

- "사람들 반응 어때?", "여론 분석해줘", "요즘 이거 분위기 어때?", "민심 어때?"
- "샤오홍슈/더우인/X에서 뭐가 트렌드야?", "중국 SNS 트렌드 알려줘", "중국 반응 어때?"
- "후기 좋아 나빠?", "이 제품/브랜드/인물/이슈 평가", "논쟁/찬반 정리해줘", "왜 싸워?"
- "요즘 뜨는 키워드", "지금 화제인 거", "곧 터질 거", "순위/랭킹/베스트 매겨줘"
- `/social-trend-research [주제]`로 명시 호출하는 경우

플랫폼을 지정하지 않아도, 사람들의 실제 의견·분위기를 묻는 질문이면 기본적으로 동작합니다. 다만 한국 네이버·인스타·유튜브만 콕 집어 요청한 경우, 영어권 매체 기반 AI 뉴스 큐레이션(news-research 스킬 영역), 단순 사실/정의 검색, 기사 1건 요약·번역, 주가·재무 분석은 대상이 아닙니다.

## 입력과 출력

무엇을 주면 무엇이 나오는지 정리하면 다음과 같습니다.

| 구분 | 내용 |
|---|---|
| 입력 | 주제(제품·브랜드·인물·이슈·키워드), 선택적으로 플랫폼·기간·분석 각도 |
| 출력 | 분석 유형에 맞는 한국어 리포트 (실제 인용 + 출처 + 직접/간접 태그 포함) |

분석 유형은 질문에 따라 6종 중 하나로 자동 선택됩니다.

| 유형 | 트리거 예시 | 산출물 |
|---|---|---|
| 빠른 답변 | "그래서 뭐가 제일 나아?" | 2~4문장 결론 + 핵심 근거 인용 |
| 랭킹 | "순위/추천/best", "뭐가 제일 많이 언급돼?" | 실제 언급 기반 순위 + 플랫폼별 차이 |
| 감성 | "반응 좋아 나빠?", "분위기 어때?" | 긍/부정/중립 분포 + 대표 인용 |
| 트렌드 | "언제부터 떴어?", "지금 오르는 중?" | 모멘텀 + 타임라인 + 스파이크 원인 |
| 논쟁 | "찬반", "왜 싸워?", "양쪽 입장" | 입장 A/B 논거 지도화 |
| 발굴 | "요즘 뜨는 거", "곧 터질 키워드" | 초기 신호 키워드 + 주도 플랫폼 |

모든 리포트는 공통으로 수집 기간, 다룬 플랫폼, 플랫폼별 실제 인용, 데이터 한계를 포함합니다. 각 인용은 다음 형식을 따릅니다.

```text
- [직접/간접] (플랫폼) "원문 짧은 인용"
  → 번역: "한국어 번역"
  출처: <링크 또는 '○○ 트렌드 리포트(2차)'>
```

원칙은 한 출처에서 인용 1개, 원문 기준 15단어 또는 30자 내외로 짧게입니다. 실제 게시물 원문을 본 경우는 `[직접]`, 트렌드 리포트·뉴스가 정리한 2차 정보는 `[간접]`으로 태깅합니다.

## 사용법

이 스킬은 [Claude](https://claude.ai) 환경에서 동작하는 사용자 스킬입니다. 별도의 API 키나 외부 MCP 의존성은 없으며, Claude의 웹 검색·웹 페치 기능을 사용합니다.

설치는 스킬 디렉터리에 폴더째 배치하면 됩니다.

```
skills/
└── social-trend-research/
    ├── SKILL.md
    └── references/
        ├── platforms.md
        └── analysis-types.md
```

등록 후에는 자연어로 요청하면 자동 트리거되거나, 슬래시 명령처럼 명시 호출할 수 있습니다.

```text
/social-trend-research 말랑이
```

또는 그냥 평범하게 묻습니다.

```text
요즘 "말랑이" 사람들 반응 어때?
샤오홍슈에서 이 브랜드 후기 좋아 나빠?
이 이슈 X에서 찬반 어떻게 갈려?
```

## 폴더 구성

```
social-trend-research/
├── SKILL.md            # 스킬 본체: 원칙·워크플로우·인용 형식
├── README.md           # 이 문서
├── LICENSE             # MIT 라이선스
├── sample.html         # 산출물 예시
└── references/
    ├── platforms.md       # 플랫폼별 데이터 접근 현실 + 검색 전략 + 신뢰도 기준
    └── analysis-types.md  # 6종 분석 유형별 한국어 출력 템플릿
```

스킬은 질문 유형이 정해지면 해당 참고 파일의 필요한 부분만 읽고 진행하도록 설계되어 있습니다.
