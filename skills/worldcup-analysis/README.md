# 월드컵 조 분석·우승확률

입력한 국가가 속한 조의 네 팀을 다섯 페르소나가 서로 다른 시선으로 분석하고, 통과 확률과 우승 확률을 산출한다.

## 무엇을 하나

최신 데이터를 검색해 조 순위와 전력 점수, 조 1·2위 통과율과 우승 확률을 표로 정리한다. 같은 수치를 비평가·열혈팬·안티팬·일반인·캐스터 다섯 페르소나가 각자 어조와 해석으로 풀어낸다. 수치 자체는 지어내지 않는다.

## 언제 쓰나

- 국가명과 함께 "스페인 어때?", "월드컵 우승확률", "OO 조 분석" 같은 요청
- 경기 직후 변동, "이기면 어때?" 같은 조건부 시나리오, 16강 대진 분석

## 입력

월드컵 본선 참가국 이름. 필요하면 특정 경기나 시나리오 조건, 기준일을 더한다.

## 산출물

- 조 순위 표 (경기수·승점·득실·FIFA 랭킹·최근 폼)
- 확률 표 (조 통과율, 우승 확률)
- 전력 점수표 (FIFA·조 성적·최근 폼·스쿼드 가중)
- 다섯 페르소나 분석

## 구성

- `SKILL.md` 페르소나 정의, 스코어링 로직, 검색 프로토콜
- `sample.html` 예시 분석

## 이런 건 하지 않는다

- 베팅 배당·도박 데이터 사용
- 미확인 항목을 추정치로 채우기
- 특정 국가나 국민에 대한 혐오 표현

월드컵 비참가국이 입력되면 먼저 알리고 멈춘다.
