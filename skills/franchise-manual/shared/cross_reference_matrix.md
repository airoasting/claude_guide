- 인용: POS 시스템·본사 ERP·가맹점주 ERP 3원 데이터 / 외부 회계 시스템
- 피인용: Ch02 (정산 마감 D+1 세금계산서) / Ch04 STEP 1 (정산 후 메뉴 원가 카드 매핑) / Ch05 STEP 1·부록(정산 정확도·미수율 KPI 통합 대시보드) / 홀서비스 Ch05 (재무팀 D+5 1원 단위 일치 검증·STEP 1~6 연계 데이터 시점 부록 3) / 조리 Ch08 (재무·정산 Ch04 STEP 1·STEP 4 환류)

#### Ch02 세무관리
- 인용: 홈택스·본사 ERP [세무관리]·POS 시스템·외부 세무 대리인 ERP
- 피인용: Ch05 STEP 1·부록 (신고 기한 100%·가산세 0건 KPI) / 인사교육 Ch06 (원천세 매월 10일 신고)

#### Ch03 자금관리
- 인용: 본사 ERP [정산관리]·[미수관리]·[자금일보] / 인터넷뱅킹 / 가맹점주 ERP / 외부 회계 시스템
- 피인용: Ch05 STEP 1·부록 (자금일보 100%·운전자금 1개월분)

#### Ch04 원가관리
- 인용: Ch01 STEP 2 매입 데이터 / 본사 ERP [원가관리]·[메뉴원가카드]·[실제원가현황] / R&D 표준 레시피 PDF / 가맹점주 ERP / 가맹운영지원 Ch02 (점포별 단일 진단 리포트)
- 피인용: Ch05 (원가율 33%·갭 ±5% KPI) / 조리 Ch08 STEP 6 (D+12 단가 환류 송부)

#### Ch05 예산관리
- 인용(자부서): Ch01·Ch02·Ch03·Ch04 (4개 도메인 통합 대시보드·5개 KPI 백서)
- 피인용: 인사교육 / 재무정산 Ch01~Ch04 (KPI 환류 단일 종결)
