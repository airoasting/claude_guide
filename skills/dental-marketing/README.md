# 치과 마케팅 벤치마킹

지역 치과들의 온라인 홍보 채널을 조사·분석해, 클라이언트에게 바로 낼 수 있는 BrandLab 브랜드 전략 제안서를 만든다.

## 무엇을 하나

지역과 진료과목을 정하면 대상 병원을 모으고, 홈페이지·블로그·플레이스·인스타그램 채널을 채널별로 조사한다. 전략 패턴과 원장 브랜딩, 수익 구조를 진단해 조사표·분석 보고서·제안서 세 가지 문서를 만든다. 제안서는 5-Color 평가 루프로 합격선까지 다듬는다.

## 언제 쓰나

- "치과 홍보 조사", "치과 마케팅 분석", "치과 SNS 벤치마킹", "BrandLab 제안서" 같은 요청
- 지역 병원 벤치마킹과 제안서가 함께 필요할 때

치과뿐 아니라 피부과·한의원·성형외과에도 쓸 수 있다.

## 입력

지역(시·구·동), 병원 수(기본 15곳), 저장 폴더. Node.js와 npm이 필요하다.

## 실행

```bash
node scripts/make_link_table.js [폴더]   # 조사표
node scripts/make_analysis.js [폴더]     # 벤치마킹 분석 보고서
node scripts/make_proposal.js [폴더]     # 제안서 (5-Color 루프 대상)
```

## 산출물

docx 세 종.

- 홍보채널 조사표
- 벤치마킹 분석 보고서
- BrandLab 브랜드 전략 제안서 (표지·요약·서비스·가격·로드맵 포함)

## 구성

- `scripts/` 문서 생성 스크립트 세 개
- `references/research-protocol.md` 조사 방법
- `references/document-specs.md` 문서 구조
- `references/5color-loop.md` 평가 기준과 중단 조건

## 이런 건 하지 않는다

- 기존 제안서의 평가·번역·요약만
- 의료와 무관한 시장 조사
- 단일 병원 한 곳 확인

의료광고법을 따르고, 가격은 협의가 아니라 범위로 명시한다.
