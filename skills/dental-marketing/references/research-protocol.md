# 리서치 프로토콜 — 병원 목록 수집 & 홍보 전략 심층 조사

이 문서는 SKILL.md의 STEP 2(병원 목록 수집)와 STEP 4(홍보 전략 심층 조사)에서 호출한다.

---

## 1. 병원 목록 수집 (STEP 2)

### 검색 전략 (WebSearch 병렬 실행)

```
[지역명] 치과 홈페이지 추천 목록 2026
[지역명] 치과 네이버플레이스 인스타그램 홍보
[지역명] 치과의원 순위 병원랭킹
```

### 보조 데이터 소스

- `https://www.goodhosrank.com` — 지역별 치과 순위
- `https://www.modoodoc.com` — 리뷰 기반 병원 목록
- `https://www.goodoc.co.kr` — 위치·진료시간 정보

### 채널별 링크 수집 방법

| 채널 | 수집 방법 |
|------|---------|
| 홈페이지 | 웹검색 + 병원랭킹 사이트 |
| 네이버 블로그 | 홈페이지 footer 링크 또는 직접 검색 |
| 네이버 플레이스 | 홈페이지 내 네이버지도 링크 또는 `map.naver.com` 검색 |
| 인스타그램 | 웹검색 `[병원명] instagram` |
| 페이스북 | 웹검색 `[병원명] facebook` |
| 유튜브 | 홈페이지 footer 또는 웹검색 |

### 수집 시 주의

- 인스타그램은 JavaScript 렌더링이 필요하다. `read_page`(interactive) 또는 웹검색으로 팔로워 수를 파악한다.
- 네이버 블로그(`blog.naver.com`)는 직접 접근 제한이 있으므로 구글 검색으로 우회한다.
- 미확인 링크는 표에 "미확인"으로 표시하고, 2차 수작업 조사 가이드를 보고서에 포함한다.

## 2. 홍보 전략 심층 조사 (STEP 4)

주요 병원(SNS 팔로워 상위 5곳 등)의 홈페이지와 인스타그램을 방문해 아래 4개 항목군을 기록한다.

### A. 기본 채널 전략
- 주력 시술명 및 독자 브랜드 여부
- 이벤트·프로모션 운영 현황
- 콘텐츠 유형 (전후사진, 교육 콘텐츠, 후기, 칼럼 등)
- 타겟 키워드 (예: "수면마취", "1일 치료", "무삭제 라미네이트")
- 신뢰도 요소 (의료진 학력, 수상, 인증 등)
- 예약 경로 (네이버예약, 카카오, 전화 등)

### B. 원장 브랜딩
- 원장 개인 SNS 또는 유튜브 채널 운영 여부
- 원장 인터뷰·칼럼·스토리 콘텐츠 존재 여부
- 병원 슬로건·BI(브랜드 아이덴티티) 일관성
- 독자 시술명 브랜딩 여부

### C. AEO (AI 검색 최적화)
- 홈페이지 FAQ 섹션 또는 Q&A 구조화 콘텐츠 존재 여부
- 구조화 데이터(Schema Markup) 적용 여부
- 구글·네이버 AI Overview 인용 여부 확인 (검색어 예: "강남 임플란트 잘하는 곳 추천")
- E-E-A-T 요소 충실도

### D. 수익 구조 진단 (BrandLab 전용)
- 주력 비급여 시술 및 가격 구조 파악 가능 여부
- 재내원·충성 환자 유도 콘텐츠 존재 여부
- 마케팅 투자 대비 수익 효율이 높은 채널 식별

### 조사 시 주의
- 의료광고법상 과장 표현을 쓰는 병원은 벤치마킹 보고서에 주의 표시를 한다.
