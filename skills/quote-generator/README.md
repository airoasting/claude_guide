# 공유용 견적서 생성

고객·제품·수량·단가를 받아 외부에 보낼 견적서 엑셀(.xlsx)을 만든다.

## 무엇을 하나

채팅으로 받은 정보를 정리해 견적서 양식을 채운다. 공급가액과 합계는 자동으로 계산하고, 항목 일부가 비어 있어도 채울 수 있다. 양식이 손상되면 템플릿을 자동으로 다시 만든다.

## 언제 쓰나

- "(주)○○에 제품 견적서 만들어줘", "자동 견적 생성" 같은 요청
- 외부 고객이나 파트너에게 표준 견적서를 보낼 때

## 입력

- 고객사: 법인명·담당자·연락처·이메일·주소·견적번호·발행일·유효기간
- 제품: 제품명·수량·단가·비고 (1~5개)
- 공급자: 선택 (기본값 있음)

정보가 부족하면 입력 양식을 먼저 보여준다.

## 산출물

`견적서_<회사명>_<YYMMDD>.xlsx`. 입력 시트와 견적서 시트로 나뉘고, 공급가액은 수량 × 단가, 합계는 자동 합산한다. VAT는 별도다.

## 구성

- `quote-light/SKILL.md` 발동 조건과 워크플로우
- `quote-light/scripts/fill_quote.py` JSON을 엑셀로 변환
- `quote-light/scripts/create_template.py` 템플릿 자동 재생성
- `quote-light/assets/quote_template.xlsx` 빈 견적서 원본
- `dist/quote-light.skill` 설치용 패키지
- `examples/` 샘플 세 종

## 이런 건 하지 않는다

- 세금계산서·거래명세서·인보이스 발행
- ERP 연동이나 내부 가격 규칙 자동 계산
- 할인 자동 적용 (단가에 반영하거나 음수 항목으로 처리)

항목은 최대 5개까지 담는다.
