# 분석 세부 명세 (analyze.py 기준)

이 문서는 `scripts/analyze.py`가 수행하는 계산을 정리한다. 수식·기간·점수표를 바꿔야 할 때 참고한다.
실제 구현이 항상 기준이며, 이 문서와 코드가 어긋나면 코드를 따른다.

## 입력과 전처리

- 파일 패턴: `data/아파트(매매)_실거래가_*.csv`. 매칭되는 파일을 **모두 읽어 concat**한다(최신 한 개만 고르지 않는다).
- 안내문 처리: 상단 15행을 건너뛴다(`skiprows=15`).
- 인코딩: `euc-kr` -> `cp949` -> `utf-8-sig` 순으로 시도, 모두 실패하면 그 파일만 건너뛴다.
- 행정동 추출: `시군구`에서 "분당구 " 뒤 문자열을 동 이름으로 사용하고, 분당구 13개 동만 남긴다.
  - 13개 동: 정자·야탑·서현·구미·수내·이매·금곡·운중·분당·판교·삼평·백현·대장
- 제외 규칙: `거래상태`에 "해제" 포함 행 제외, `거래금액`·`전용면적` 결측 또는 면적 0 이하 제외.

## 평당가

```
평당가(만원/평) = 거래금액(만원) / (전용면적(㎡) / 3.3058)
```

`PYEONG_FACTOR = 3.3058` (1평 = 3.3058㎡).

## 변동률 (전반기 vs 후반기)

- 전반기: 2025.06.15 ~ 2025.12.13
- 후반기: 2025.12.14 ~ 2026.06.14
- 동별·단지별로 각 기간 평균 평당가를 구해 `(후반기 - 전반기) / 전반기 × 100`, 소수점 1자리.
- 두 기간 중 한쪽이라도 거래가 없으면 해당 그룹은 제외(dropna).

## 전년 동기비 (콘솔 출력용)

- 당해: 2025.06.15 ~ 2026.06.14
- 전년: 2024.06.15 ~ 2025.06.14
- 동별 평균 평당가 비교. PDF에는 들어가지 않고 콘솔에만 출력된다.

## 이 달의 거래 (highlight)

- 각 동의 평균 변동률과 단지 변동률의 편차(절댓값)를 구한다.
- 동별로 편차 최대 단지를 1개씩 뽑고, 전체를 편차 내림차순 정렬.
- PDF는 상위 5개를 표시.

## 단지 종합점수 (100점 만점)

| 항목 | 만점 | 산정 방식 |
|---|---|---|
| 입지 | 40 | 동별 기준표(`DONG_BASE_SCORES`)의 입지 점수 |
| 평형/구조 | 20 | 전용면적 중위값 구간별(`_score_area`): >=135->18, >=100->17, >=85->16, >=60->13, 그 외 10 |
| 학군 | 15 | 동별 기준표의 학군 점수 |
| 재건축/개발 | 10 | 건축년도 노후도(`_score_reconstruction`, 2026 기준): >=35년->10, >=30->8, >=25->6, >=20->4, >=10->2, 그 외 0 |
| 단지 규모 | 10 | 전체 거래건수 프록시(`_score_size`): >=150->9, >=80->8, >=40->6, >=20->4, >=10->3, 그 외 2 |
| 동·층·조망 | 5 | 개별 거래로 산정 불가, 기본값 3 |

- 동별 기준표에 없는 동은 입지 28·학군 9를 기본으로 쓴다.
- `scores_override.json`에 `"행정동_단지명": {"종합점수": N}`이 있으면 그 단지는 합산 대신 N으로 덮어쓴다.
- 합산 결과는 100으로 상한.

## 주목할 단지 (저평가)

```
점수당가격 = 평당가_전체평균 / 종합점수    # 낮을수록 저평가
```

- `--min-deals`(기본 5) 미만 거래 단지는 후보에서 제외(소량 거래 이상치 방지).
- 동별 1위(`dong_notable`)와 분당구 전체 TOP N(`top_notable`)을 산출. PDF는 전체 TOP 5를 표시.

## run() 인터페이스

```python
from analyze import run
results = run(data_dir, min_deals=5, top_n=5)  # dict 반환
```

반환 dict 키: `df, dong_rate, complex_rate, highlight, scored, dong_notable,
top_notable, monthly_trend, monthly_dong, monthly_complex, reasons, yoy_rate`.
`pdf_report.py`는 이 중 `df, dong_rate, highlight, top_notable`을 사용한다.

## 대출 기준 (인사이트 단락에 인용)

투기과열지구·토지거래허가구역(2025.10.16~) 기준.

- 매매가 15억 이하 -> 최대 대출 6억
- 15억 초과 ~ 25억 이하 -> 4억
- 25억 초과 -> 2억
- 최소 자기자본 = 매매가 - 대출한도 + 부대비용(약 1.5%)
