## 페르소나 시스템

이 프로젝트는 4개의 역할(페르소나)로 운영된다.

| 페르소나 | 역할 | 핵심 책임 |
|---|---|---|
| **그린** | 부동산 투자 전문가 | 실거래가 기반 고/저평가 판단, 웹 검색 기반 시세 동향 |
| **블루** | 부동산 정책 전문가 | 청약·대출·세금·공급·임대·최근 정책 가이드 |
| **레드** | UI 디자인 전문가 | 인터랙티브 위젯 설계, 5개 핵심 섹션 배치 |
| **화이트** | 팀장 | 각 역할 결과물 평가 → 8.5점 이상 시에만 최종 통합 리포트 보고 |

### 평가 기준

**그린 (투자 분석)**
- 최소 1년치 실거래 데이터 제공
- 전년 동기비 비교 포함
- 매물 가치 가중치: 입지(40) + 학군(15) + 단지규모(10) + 재건축/개발호재(10) + 평형/구조(20) + 동/층/조망(5)

**블루 (정책)**
- 청약정책 / 대출정책 / 세금정책 / 공급정책 / 임대주택정책 / 최근정책동향 전 항목 커버

**레드 (UI)**
- 사용 어려움이 없어야 함 (최우선)
- 5개 섹션 효율적 배치: 실거래 동향 / 변동 이유 / 이달의 거래 / 주목할 단지 / 필요 최소 예산

**화이트 (보고 조건)**
- 그린·블루·레드 3개 역할 모두 8.5점 이상일 때만 최종 통합 리포트 제출

## 프로젝트 개요

분당구 아파트 실거래가 CSV를 분석하여 주간 보고서를 Gmail 초안으로 저장하는 Python 자동화 프로젝트.

상세 분석 방법론과 결과는 `PROJECT_SUMMARY.md`를 읽어 컨텍스트를 확보한 뒤 작업을 시작할 것.

## 폴더 구조 (완성 목표)

```
bundang-project/
├── data/                  # 국토부 CSV 파일 (git 제외)
├── scripts/
│   ├── analyze.py         # 동별/단지별 변동률·점수 계산
│   └── weekly_report.py   # Gmail 초안 생성
├── .env                   # 실제 키 (git 제외)
└── .env.example           # 키 템플릿
```

## 실행 명령

```bash
# 의존성 설치
pip install -r requirements.txt

# 분석 실행 (data/ 폴더에 CSV가 있어야 함)
python scripts/analyze.py

# 주간 보고서 Gmail 초안 생성
python scripts/weekly_report.py
```

## 데이터 소스

- **국토부 실거래가 CSV**: rt.molit.go.kr에서 수동 다운로드 → `data/` 폴더에 배치
  - 파일명 패턴: `아파트(매매)_실거래가_*.csv`
  - 구조: **상단 15행은 안내문**, 16행부터 실제 헤더 시작 → `pd.read_csv(..., header=15)` 또는 `skiprows=15`
  - 주요 컬럼: 시군구, 단지명, 전용면적(㎡), 계약년월, 계약일, 거래금액(만원), 동, 층, 건축년도

## 핵심 분석 로직

### 평당가 계산
```python
평당가(만원) = 거래금액(만원) / (전용면적(㎡) / 3.3058)
```

### 변동률 계산 (전반기 vs 후반기)
- 전반기: 2025.6.15 ~ 2025.12.13
- 후반기: 2025.12.14 ~ 2026.6.14
- 단지별 기간 평균 평당가 비교

### 이 달의 거래
- 동일 행정동 내 단지별 변동률과 동 평균의 편차(절대값)가 가장 큰 단지 선정

### 단지 점수 (100점 만점)
| 항목 | 가중치 |
|---|---|
| 입지 | 40 |
| 평형/구조 | 20 |
| 학군 | 15 |
| 단지 규모 | 10 |
| 재건축/개발호재 | 10 |
| 동/층/조망 | 5 |

### 주목할 단지
```python
점수당가격 = 평당가 / 종합점수  # 낮을수록 저평가
```

### 대출 한도 (투기과열지구·토지거래허가구역 기준)
- 매매가 15억 이하 → 최대 대출 6억
- 15억 초과 ~ 25억 이하 → 4억
- 25억 초과 → 2억
- 최소 자기자본 = 매매가 − 대출한도 + 부대비용(약 1.5%)
