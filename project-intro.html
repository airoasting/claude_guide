Step 1: 프로젝트 소개

프로젝트란 무엇인가?

개념을 한 문장으로 이해하기
한 줄 정의

프로젝트 = 나만의 전용 AI 업무 공간
한 번 설정해두면, 이후 모든 대화에서 Claude가 그 맥락을 자동으로 기억합니다.

팀장이 "이번 주 보고서 정리해줘"라고 요청합니다. Claude를 열고 또다시 입력합니다. "우리 회사 어조는 격식체고, A4 2장 분량으로, 핵심 수치 먼저 배치하고, 소제목은 굵게…" 매주 같은 3분을 반복하다 보면, 정작 중요한 데이터를 빠뜨리기도 합니다.

프로젝트는 이 반복을 없애줍니다. 업무별로 파일과 규칙을 한 번만 저장해두면, 그 프로젝트 안에서는 새 대화를 시작해도 Claude가 모든 맥락을 유지합니다. 매주 반복하던 컨텍스트 입력 시간이 사라지고, 응답 품질도 처음부터 일정하게 유지됩니다.

✕ 프로젝트 없을 때

매주 보고서 쓸 때마다
"격식체로, A4 2장 분량, 핵심 수치 먼저, 팀장 보고용…"
처음부터 다시 설명

✓ 프로젝트 사용할 때

"이번 주 실적 요약해줘"
한 마디면 끝.

핵심 구분: 메모리 vs 지침 vs 파일

세 가지가 비슷해 보이지만 역할이 완전히 다릅니다. 아래 표로 한눈에 확인하세요.

Step 3: 시스템 프롬프트

시스템 프롬프트란?

Instructions의 정체

시스템 프롬프트(System Prompt)는 Claude에게 "항상 이렇게 행동해줘"라고 미리 지시하는 텍스트입니다. Claude.ai 프로젝트에서는 Project Instructions라는 이름으로 설정할 수 있습니다.

사용자가 새 대화를 시작할 때마다 이 지시사항이 자동으로 Claude에게 전달됩니다. 덕분에 "한국어로 답해줘", "격식체 사용해줘" 같은 말을 매번 반복하지 않아도 됩니다.

✍️ Instructions 작성 역할·형식·금기를 한 번만 입력
→
💬 새 대화 시작 Claude가 자동으로 Instructions 수신
→
⚡ 즉시 적용 설명 없이 바로 원하는 결과 출력
→
🔁 무한 반복 프로젝트 내 모든 대화에서 동일하게
Step 4: 작성 가이드

잘 쓴 시스템 프롬프트의 4가지 요소

이 순서로 작성하면 Claude가 정확히 움직입니다
🎭 역할 (Role) Claude가 맡을 역할을 명확히 정의합니다.
"너는 10년 경력의 마케팅 전문가야"
🗣️ 말투 (Tone) 응답의 격식 수준과 스타일을 지정합니다.
"친근한 반말로, 이모지 없이"
📐 형식 (Format) 출력 구조를 지정합니다.
"항상 요약 → 본문 → 다음 액션 3가지 순서로"
🚫 금기사항 (Constraints) 하지 말아야 할 것을 명시합니다.
"영어로 답하지 말 것, 불확실한 정보는 추측하지 말 것"

좋은 예 vs 나쁜 예

구체성이 퀄리티를 만듭니다
✗ 모호한 지시

"전문적으로 답해줘.
좋은 내용으로 작성해줘.
도움이 되는 AI가 되어줘."

✓ 구체적인 지시

"너는 화장품 브랜드 회사의 마케팅 팀장이야. 한국어로, 존댓말로, 500자 이내로 답해줘."

💡
핵심 원칙

Claude는 명확할수록 잘 따릅니다. "좋게" 대신 "300자 이내로", "전문적으로" 대신 "서울대 교수가 학부생에게 설명하듯"처럼 구체적으로 쓰세요.

Step 5: 템플릿 & FAQ

복사해서 바로 쓰는 템플릿

빈칸만 채우면 완성
📋 업무 보조 템플릿
너는 나의 업무 보조 AI야.

[역할]
- 10년 경력 비즈니스 컨설턴트처럼 답해줘
- 실용적이고 바로 실행 가능한 내용 중심으로

[말투]
- 한국어, 존댓말, 간결하게
- 불필요한 서론 없이 핵심부터

[형식]
- 긴 답변은 반드시 소제목으로 구분
- 마지막에 "다음 액션" 1~3가지 제안

[금기]
- 확실하지 않은 사실은 추측하지 말 것
- "물론입니다", "좋은 질문입니다" 같은 빈말 금지
📋 이메일 작성 템플릿
너는 나의 이메일 작성 도우미야.

[컨텍스트]
- 나는 IT 스타트업에서 일하는 PO야
- 주로 외부 파트너사, 고객사와 소통해

[이메일 규칙]
- 격식체 (하십시오체) 사용
- 제목: 목적이 명확하게 (예: [요청] 미팅 일정 조율 건)
- 본문: 인사 → 용건 → 요청사항 → 마무리
- 200자 내외로 간결하게
- 서명: "감사합니다. [이름] 드림" 형식
📋 콘텐츠 기획 템플릿
너는 나의 SNS 콘텐츠 기획자야.

[브랜드 정보]
- 브랜드: AI 생산성 도구 스타트업
- 타겟: 30~40대 직장인, 비개발자
- 톤: 친근하지만 신뢰감 있게

[콘텐츠 규칙]
- 첫 문장은 공감대 형성 (문제 제기)
- 이모지 최대 3개
- 해시태그 5개 이하
- 링크드인 기준으로 작성

→
3️⃣ Instructions 수정 "친근한 존댓말" 추가
→
4️⃣ 재요청 개선된 결과 확인

자주 묻는 질문 & 알아두면 좋은 점

실전에서 자주 부딪히는 궁금증 해결
▶ 무료 계정으로도 프로젝트를 만들 수 있나요?

네, 무료 계정에서도 프로젝트 생성이 가능합니다. 다만 일일 메시지 수 제한이 있고, Opus 등 고성능 모델은 Pro 요금제(월 $20)에서 사용할 수 있습니다.

▶ Instructions를 너무 길게 쓰면 문제가 되나요?

Instructions도 Claude의 컨텍스트 윈도우를 사용합니다. 너무 길면 실제 대화 공간이 줄어듭니다. 핵심 규칙 위주로 300~500자 이내가 가장 효율적입니다.
