크롬에서 클로드 플러그인을 써야 하는 이유

왜 크롬에서 Claude를 써야 할까?

당신의 업무 80%는 이미 크롬 안에서 일어나고 있습니다

이메일 작성, 뉴스 확인, 자료 검색, 보고서 리뷰. 이 모든 순간이 크롬 안에서 일어납니다. 그런데 왜 AI를 쓰려면 매번 다른 탭으로 이동해서 복사-붙여넣기를 해야 할까요?

크롬 플러그인을 설치하면, 지금 보고 있는 바로 그 페이지 위에서 Claude가 일합니다.

🖱️

실제 클릭·입력까지 대행

단순 요약이 아닌 폼 작성, 버튼 클릭, 다중 탭 이동 등 사람이 하던 조작을 그대로 수행합니다.

🎬

녹화 한 번 → / 명령어로 무한 반복

반복 업무를 한 번만 시연하면 매크로가 자동 생성. "/주간보고" 한 줄로 매주 똑같은 작업을 재실행합니다.

⏰

시간 예약 실행

"매주 월요일 오전 9시" 같이 스케줄을 걸어두면 브라우저가 알아서 로그인·수집·정리·발송까지 끝냅니다.

설치

설치 & 초기 설정

지금 바로 따라 해보세요. 30초면 끝납니다

🧩 Chrome 웹스토어에서 Claude 설치하기
  1. 위 버튼 클릭
    Chrome 웹스토어의 Anthropic 공식 페이지가 열립니다. Claude for Chrome 확장 프로그램을 찾아 클릭합니다.
  2. "Chrome에 추가" 클릭
    파란 버튼을 누르고, 팝업에서 "확장 프로그램 추가"를 누르면 바로 설치됩니다.
  3. Claude 계정으로 로그인
    주소창 오른쪽에 생긴 Claude 아이콘을 클릭하면 로그인 화면이 나타납니다. 기존 계정으로 로그인하거나 새로 가입합니다.
  4. 아이콘 고정하기
    주소창 오른쪽의 퍼즐 아이콘()을 클릭하고, Claude 옆의 핀 아이콘을 눌러 항상 보이게 고정합니다.
  5. 테스트
    아무 웹 페이지에서 Claude 아이콘을 클릭하고 "이 페이지를 한 줄로 요약해줘"라고 입력합니다. 답변이 오면 설치 완료!
💡 설치 후 이렇게 보여요

설치가 완료되면 Chrome 주소창 오른쪽에 아이콘이 생깁니다. 가운데 버튼이 Claude 확장 프로그램입니다.

핵심 기능

핵심 기능 5가지

이런 것까지 된다고? 한번 써보면 놀라는 기능들

📰

1. 웹 페이지 요약

긴 기사, 보고서, 블로그 글의 핵심만 3줄로. 리서치 시간이 절반으로 줄어듭니다.

🌍

2. 실시간 번역

구글 번역과 다릅니다. 맥락을 이해하는 자연스러운 한국어 번역을 제공합니다.

✉️

3. 이메일 작성 보조

Gmail에서 바로 답장 초안 생성. 톤, 길이, 목적에 맞게 작성합니다.

🔍

4. 검색 결과 분석

여러 탭을 열지 않아도 됩니다. 검색 결과를 한 번에 비교 분석합니다.

📝

5. 텍스트 선택 후 즉시 작업

드래그한 텍스트만 바로 요약·번역·톤 변경·핵심 추출. 필요한 부분만 정확하게 처리합니다.

실전 활용

실전 활용 시나리오 6가지

"이걸 이렇게 쓴다고?" 내일 당장 써먹는 상황별 가이드

📌 시나리오 1: 아침 뉴스 브리핑

매일 아침 뉴스 사이트에서 주요 기사를 열고 Claude에게 요약을 요청합니다.

이 기사의 핵심 내용을 3가지 포인트로 정리해줘. 우리 회사 마케팅팀에 공유할 수 있는 한 줄 요약도 추가해줘.

📌 시나리오 2: 영문 리포트 빠른 파악

해외 파트너가 보내준 영문 보고서를 번역하면서 핵심 수치를 추출합니다.

이 페이지를 한국어로 번역해줘. 특히 매출 수치와 성장률은 표로 정리해줘.

📌 시나리오 3: 이메일 답장 초안

Gmail에서 받은 메일을 보면서 바로 답장 초안을 생성합니다.

이 이메일에 대한 답장을 작성해줘. 정중하지만 일정 변경은 어려움을 알려주는 톤으로, 대안 일정 2개를 제안해줘.

📌 시나리오 4: 경쟁사 분석

경쟁사 웹사이트를 방문하면서 주요 특징을 분석합니다.

이 웹사이트의 주요 서비스, 가격 정책, 차별화 포인트를 정리해줘. 우리 서비스와 비교할 수 있는 표 형태로 만들어줘.

📌 시나리오 5: 회의 자료 빠른 리뷰

Google Docs나 Notion에서 공유된 회의 자료를 빠르게 파악합니다.

이 문서의 핵심 결정 사항과 액션 아이템을 정리해줘. 내가 담당해야 할 항목이 있으면 별도로 표시해줘.

📌 시나리오 6: SNS 콘텐츠 작성

참고할 글이나 기사를 보면서 SNS 포스팅을 바로 생성합니다.

이 기사 내용을 바탕으로 LinkedIn 포스팅을 작성해줘. 전문적이면서도 인사이트가 있는 톤으로, 300자 내외로 만들어줘.
실습 과제

7가지 실습 과제

크롬 브라우저만 있으면, 지금 바로 AI 비서를 고용할 수 있습니다

각 과제를 클릭하면 상세 안내가 펼쳐집니다. 쉬운 것부터 도전해보세요!

1 긴 텍스트 페이지를, 3줄로 요약하기 ★

관심 있는 뉴스 기사를 크롬에서 열고, Claude 사이드패널에 입력합니다:

"이 페이지 내용을 3줄로 요약해줘. 이 내용이 중요한 이유를 한 줄로 알려줘."

💬 후속 질문도 해보세요: "이 기사의 주장에 반론을 제기한다면?"

2 웹사이트 3개를 동시에 비교 · 분석하기 ★★

탭 3개를 열어 비교할 웹사이트에 각각 접속합니다. 그런 다음 사이드패널에 아래 프롬프트를 입력합니다:

  1. Ctrl+T (Mac: Cmd+T) 로 탭 3개 열기
  2. 각 탭에서 비교할 웹사이트 접속
  3. Claude 사이드패널에 아래 프롬프트 입력
"이 3개 탭의 내용을 비교해줘. 공통점과 차이점을 테이블로 정리해줘."
3 AI가 내 대신 클릭하고 입력합니다 ★★

Google Flights(flights.google.com)를 열고, 사이드패널에 입력합니다:

"서울에서 도쿄로 가는 다음 달 왕복 항공권을 검색해줘. 가장 저렴한 옵션 3개를 알려줘."

👀 Claude가 출발지·도착지·날짜를 직접 입력하고 검색 버튼을 클릭하는 것을 지켜보세요.

4 로그인한 사이트에서도 일합니다 ★★★

Google Calendar(calendar.google.com)를 열고, 사이드패널에 입력합니다:

"이번 주 내 일정을 확인하고, 회의가 몇 개인지 알려줘."

💡 캘린더를 안 쓰면 Gmail에서 "읽지 않은 메일 중 중요한 것 5개만 골라줘"로 해도 됩니다.

6 한 번 보여주면, 다음부턴 혼자 합니다 ★★★★

내가 하는 작업을 Claude에게 한 번 보여주면, 다음부턴 버튼 하나로 반복합니다. 사이드패널 입력창에 /를 입력하면 Record workflow 메뉴가 나옵니다.

  1. 입력창에 / 입력 → Record workflow 선택
  2. 평소처럼 웹 작업 수행 (예: 네이버 뉴스 검색, Gmail 확인)
  3. 녹화 종료 → 숏컷 자동 생성
  4. 다음부턴 / 입력 후 숏컷 선택하면 자동 실행

🎙️ 녹화 중 "지금 뉴스 기사 클릭하고 있어" 처럼 말하면서 작업하면 더 정확합니다.

7 시키지 않아도 매일 아침 출근합니다 ★★★★

과제 6의 숏컷에 실행 시간을 붙이는 단계. 같은 포맷으로 매일 쌓이기 때문에 시간 변화를 추적할 수 있습니다.

  1. 사이드패널 오른쪽 상단 ⋮ 버튼 클릭
  2. 🕐 작업으로 변환 선택
  3. 과제 6의 숏컷 선택 → 실행 시간 설정 (예: 매일 오전 9:00)
🔧 먼저 할 일: 프롬프트 속 {우리 회사명}, {경쟁사 A}, URL 등 중괄호 표시는 본인 환경에 맞게 교체하세요. 그대로 복사하면 원하는 결과가 안 나옵니다.
📬 결과를 어디서 받을지 미리 정하세요. 자동 실행된 결과는 기본적으로 사이드패널에 남지만, 각 프롬프트 끝에 이 한 줄을 붙이면 원하는 곳으로 전달됩니다:
· 이메일로: "완료되면 요약을 내 Gmail(me@...)로 보내줘"
· 슬랙으로: "결과를 슬랙 #daily-brief 채널에 붙여줘" (슬랙 탭 열려있어야 함)
· 축적용: "결과를 Google Docs '일일 브리핑'에 오늘 날짜 추가해 append 해줘" → 주간·월간 변화 추적 가능

고급 팁 & FAQ

고급 활용 팁

한 단계 더. 전문가처럼 크롬 플러그인을 활용하는 방법

🎯 단축키 설정

Chrome 주소창에 chrome://extensions/shortcuts를 입력하면 단축키 설정 페이지가 열립니다. Claude 항목에서 원하는 단축키를 지정하면 키보드 하나로 사이드패널을 열 수 있습니다. (예: Mac ⌘+E, Windows Ctrl+E)

📌 자주 쓰는 프롬프트는 과제 6의 숏컷으로

반복해서 쓰는 질문은 과제 6의 Record workflow로 숏컷을 만들어 두세요. "/" 입력 후 클릭 한 번으로 불러올 수 있습니다. 메모장에 프롬프트를 모아두는 것보다 훨씬 편합니다.

⚠️ 주의사항

금융 정보, 개인정보가 포함된 페이지에서는 Claude에 전송되는 내용을 반드시 확인하세요. 사내 보안 정책에 따라 특정 사이트에서는 확장 프로그램 사용이 제한될 수 있습니다.

FAQ

자주 묻는 질문

처음 쓰는 분들이 가장 많이 묻는 질문 5가지

Q. 무료로 사용할 수 있나요?

아니요. Claude in Chrome은 유료 플랜(Pro, Max, Team, Enterprise)에서만 사용 가능합니다. 무료 플랜에서는 확장 프로그램에 접근할 수 없습니다.

Q. 회사 컴퓨터에서도 설치할 수 있나요?

회사 IT 정책에 따라 다릅니다. Chrome 관리형 브라우저를 사용하는 경우, IT팀에 확장 프로그램 허용을 요청해야 할 수 있습니다.

Q. 내가 보는 웹 페이지 내용이 저장되나요?

Claude에 전송한 내용은 대화 기록으로 저장될 수 있습니다. 민감한 정보는 전송 전에 확인하세요. Anthropic의 개인정보 처리방침을 참고하시기 바랍니다.

Q. 다른 브라우저에서도 사용할 수 있나요?

Chrome 및 Chromium 기반 브라우저(Edge, Brave, Arc 등)에서만 사용 가능합니다. Safari, Firefox에서는 사용할 수 없습니다.

Q. 오프라인에서도 작동하나요?

아닙니다. Claude는 클라우드 기반이므로 인터넷 연결이 필요합니다.
