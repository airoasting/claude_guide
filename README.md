## 버전 히스토리

GitHub 첫 커밋은 2026년 2월 24일에 올렸습니다(v1.0). 그 뒤로는 매주 일요일마다 그 주 작업을 정리해 0.1씩 버전을 올리고 있습니다.

- **v3.0 · 2026-07-12: 다른 콘텐츠에 Hound 추가와 카드 정리**
  메인 '다른 콘텐츠'에 답을 찾을 때까지 채널을 옮겨가며 끝까지 추적하는 검색 스킬 Hound를 더했습니다. 카드 배열을 첫 줄 6개, 둘째 줄 5개로 다시 잡고 잘리던 제목을 간결하게 정리했습니다(AI 뉴스레터·5 Color·25인 자문단·Korean).
- **v2.9 · 2026-07-05: 다른 콘텐츠 개편과 앤트로픽 소개 최신화**
  메인 '다른 콘텐츠' 링크를 10선으로 다시 골라 한국어 윤문 스킬과 GPT 이미지 프롬프트 랩을 더했습니다. 실전 예제의 기본 예제 세 과제는 백업으로 내려 구성을 가볍게 했습니다. 앤트로픽 소개(엿보기)에 6월 말 소네트 5·클로드 사이언스 공개와 페이블 5 재공개까지 반영하고, 히어로의 최신 모델 표기를 Fable 5·Opus 4.8·Sonnet 5로 갱신했습니다.
- **v2.8 · 2026-06-28: 스킬 라이브러리 확장과 라이선스 정리**
  수강생이 직접 만든 스킬을 더해 스킬 쇼케이스를 21선으로 늘리고, 전 스킬의 README와 라이선스를 MIT로 통일하며 샘플의 실명과 연락처를 익명화했습니다. 검증 트랙(동조·환각)에 1차 출처로 교차검증한 실제 사례를 보강하고, 용어 사전을 73선으로 확장했습니다.
- **v2.7 · 2026-06-21: 쇼케이스 보강**
  수강생들이 직접 만든 결과물을 모은 수강생 쇼케이스와, 전 세계 클로드 코드 해커톤 우승작 14선을 모은 클로드 해커톤 쇼케이스를 새로 정리하고 보강했습니다. 실제 사례로 무엇까지 가능한지 한눈에 보도록 구성했습니다.
- **v2.6 · 2026-06-14: 루프 엔지니어링 트랙 + 보안·법률 섹션**
  행동하고 결과를 관찰하고 조정하기를 목표가 달성될 때까지 반복하는 '루프 엔지니어링' 트랙을 새로 깔고, 루프 엔지니어링과 Routines 예약 실행 두 페이지를 더했습니다. 보안 가이드와 인공지능기본법 한눈에를 '보안·법률' 섹션으로 묶었습니다.
- **v2.5 · 2026-06-07: 보안 가이드 + 하네스 엔지니어링 정비**
  코드를 모르는 리더도 바로 챙길 수 있는 'AI 협업 보안 5가지' 페이지를 새로 깔았습니다. 자동화 트랙의 하네스 엔지니어링 카드를 3단으로 정리하고, 도구 다섯 종류를 한 지도에 담은 `claude-tools.html`을 더했습니다.
- **v2.4 · 2026-05-31: 한국 법령 MCP 예제**
  Claude Desktop을 법제처 Open API에 물려, "이 조항 찾아줘" 한마디로 실제 법령을 끌어오는 흐름을 실전 예제에 더했습니다.
- **v2.3 · 2026-05-24: 용어 사전 6막 재구성**
  AI 70년사를 6막 60선의 이야기로 다시 짰습니다. 외우는 사전이 아니라 흐름으로 읽는 지도로 바꿨습니다.
- **v2.2 · 2026-05-17: 표준화**
  페이지마다 들쭉날쭉하던 본문 폭과 헤더 메뉴를 하나로 맞추고, 한국어 작성 원칙과 운영 문서 체계를 정착시켰습니다.
- **v2.1 · 2026-05-10: 공유 미리보기 + 강의 자료 보강**
  링크를 붙여넣으면 뜨는 미리보기 카드(메타 태그)를 정돈하고, 강의에 바로 쓰는 슬라이드와 예시를 보강했습니다.
- **v2.0 · 2026-05-03: 모바일 최적화 + 첫 화면 리뉴얼**
  손안에서도 막힘없이 읽히도록 모바일을 전면 손보고, 첫 화면과 안내 모달을 새로 짰습니다.
