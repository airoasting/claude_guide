## 버전 히스토리

GitHub 첫 커밋은 2026년 2월 24일에 올렸습니다. 버전 규칙은 다음과 같습니다.

- **메이저(앞 숫자, 1.x.x):** 사이트 전체 표준화나 큰 방향 전환.
- **마이너(가운데 숫자, x.1.x):** 새 페이지 추가, 새 기능 도입, 큰 디자인 변경.
- **패치(끝 숫자, x.x.1):** 작은 수정, 문구 정리, 부분 보강.

### v1.0.0 (2026-05-17) · 운영 체계 정착

사이트의 코드와 글을 한 기준 아래로 묶은 메이저 릴리스입니다. 그동안 페이지마다 흩어져 있던 디자인 토큰과 문체를 단일 문서로 모아 앞으로의 확장이 흔들리지 않을 토대를 만들었습니다.

- 모든 페이지의 헤더 메뉴 폭을 메뉴 개수별 기준 페이지에 정렬했습니다. 2-menu는 `claude-orientation.html`, 3-menu는 `ai-fluency.html`이 표준이 됩니다.
- 작업 지침을 한 곳에 모은 운영 문서 체계를 도입했습니다. `AGENTS.md`가 단일 출처이고, `CLAUDE.md`는 진입점, `MEMORY.md`는 변경 이력을 맡습니다.
- 한국어 작성 원칙 8가지(em dash 금지, 주술 구조 맞춤, 번역투 거두기, 자기해설 금지, 과장 어휘 절제, 영어 병기 최소화, 종결체 통일, AI 마무리 명언 금지)를 명문화해 콘텐츠와 운영 문서가 같은 톤으로 쓰이도록 했습니다.
- 헤더에서 본문과 중복되는 정보를 정리해 첫인상의 군더더기를 줄였습니다.

### v0.11.0 (2026-05-17) · 사이트 전면 표준화

콘텐츠 페이지 29개의 골격을 단일 규격으로 정렬한 메이저 릴리스입니다. 이때부터 새 페이지를 만들어도 다른 페이지와 시각적으로 충돌하지 않습니다.

- 본문 폭을 1080px로, 좌우 패딩과 하단 여백을 32px·80px로 통일해 페이지마다 들쭉날쭉하던 폭 정의를 단일 토큰으로 모았습니다.
- 모든 페이지의 모바일 메뉴 동작을 한 패턴으로 다듬어 페이지를 옮길 때 흐름이 끊기지 않게 만들었습니다.
- 구글 시트 대시보드 가이드를 라이브 연동 흐름으로 다시 썼고, 피자 프랜차이즈 대시보드도 뉴모피즘과 CSV 직접 fetch 방식으로 개편해 실전 예제 트랙의 톤을 일치시켰습니다.
