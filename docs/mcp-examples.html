MCP 연결 실전

외부 도구를 MCP로 연결하는 실전 예제 다섯 가지. 연결 경로와 복사용 주소까지 이 페이지에서 바로 따라 합니다

1 구글 드라이브 문서·시트 2 카카오톡 PlayMCP 나에게 보내기 3 기업 공시 분석 DART 리포트 4 한국 법령 검색 법령·판례 5 텔레그램 주고받기 봇·CLI
구글 드라이브 연결하기
⏱ 8분입문Claude Desktop · 커넥터
▼

구글 드라이브를 Claude에 커넥터로 붙여, 드라이브의 문서·스프레드시트·PDF를 Claude가 직접 찾아 읽습니다. 파일을 일일이 붙여넣지 않아도 "내 드라이브에서 ○○ 찾아줘"로 바로 씁니다.

핵심 스텝

  1. Claude Desktop을 열고 사용자 지정(Customize) → 커넥터 → 커넥터 둘러보기로 들어갑니다(아래 그림).
  1. 검색창에 Google Drive를 입력해 나온 카드에서 + 추가를 누르고 연결을 누릅니다.
  2. 열린 창에서 Google 계정으로 로그인하고 접근 권한에 동의합니다.
  3. 새 대화에서 아래처럼 물어보면 연결을 확인할 수 있습니다.

테스트 질문

내 구글 드라이브에서 지난달 회의록 문서를 찾아 3줄로 요약해줘.

회사 구글 계정이면 관리자가 커넥터를 열어두어야 합니다.

카카오톡 PlayMCP 연결하기
⏱ 12분중급Claude Desktop
▼

먼저 playmcp.kakao.com에서 회원가입하고 도구함에 쓸 도구를 담은 다음, Claude Desktop에 PlayMCP 커넥터를 붙입니다. 그러면 작업 요약을 카카오톡 나와의 채팅방으로 보내고, PlayMCP 하나로 검색·지도·유튜브·공시 도구까지 함께 씁니다.

핵심 스텝

  1. playmcp.kakao.com에 접속해 카카오 계정으로 로그인합니다. 카카오톡을 쓰는 계정이면 회원가입 없이 바로 들어갑니다.
  2. 로그인하면 도구함이 보입니다. 여기에 쓸 도구를 먼저 담아 둡니다. 아래 목록을 반드시 포함합니다.
🧰 STEP 2에서 도구함에 반드시 넣을 도구
  • 카카오톡 나와의 채팅방으로 요약과 알림을 나에게 보냅니다.필수
  • 네이버 검색으로 뉴스와 블로그, 웹 문서를 실시간으로 찾습니다.
  • 카카오맵으로 장소를 찾고 길을 안내받습니다.
  • YouTube Data로 영상과 채널, 자막, 통계를 분석합니다.
  • OpenDART로 기업 공시와 재무 데이터를 조회합니다.
  • 법률절차 길잡이로 상황에 맞는 법률 절차를 안내받습니다.
  1. Claude Desktop을 열고 왼쪽 사이드바에서 사용자 지정(Customize)을 누릅니다(아래 그림). 그다음 커넥터 → 커넥터 둘러보기로 들어갑니다.
  1. 검색창에 playmcp를 입력해 나온 카드에서 + 추가를 누르고 연결을 누릅니다.
  2. 연결 창에서 카카오 계정으로 로그인하고 권한에 모두 동의합니다. 이러면 준비가 끝납니다.
  3. 새 대화에서 아래 테스트 메시지를 보내 잘 도착하는지 확인합니다.

기업 공시 데이터 분석
⏱ 12분중급Claude Code · DART 스킬
▼

GitHub에서 DART 공시 분석 스킬을 내려받아 Claude Code에 설치하면, /dart 종목명 한 줄로 그 기업의 재무와 공시를 분석한 HTML 리포트를 받습니다. 13인 투자자 페르소나 평가까지 담깁니다.

dart 스킬 · github.com/airoasting/dart ›
이 예제는 터미널에서 claude로 켠 Claude Code 안에서 진행합니다. 설치는 자연어로 부탁하면 Claude가 알아서 받아 설치합니다.

핵심 스텝

  1. Claude Code에서 아래처럼 자연어로 스킬 설치를 부탁합니다. GitHub 주소를 받아 Claude가 알아서 내려받고 설치합니다.

스킬 설치 (Claude Code에 그대로 입력)

https://github.com/airoasting/dart 다운로드 받은 다음, 클로드 스킬로 설치해주세요.
  1. opendart.fss.or.kr에서 무료 40자 API 키를 받아 프로젝트 .env에 넣습니다.

.env에 API 키 등록

# 프로젝트 루트/.env DART_API_KEY=발급받은_40자_키_붙여넣기
  1. Claude Code에서 /dart 종목명을 실행하면 output/ 폴더에 HTML 리포트가 만들어집니다.

리포트 생성

/dart 카카오

종목명만 바꾸면 어떤 상장사든 리포트가 나옵니다. 분기도 지정할 수 있습니다. 예: /dart 삼성전자 1Q2026. 만든 리포트를 매일 아침 자동으로 받고 싶으면 아래 텔레그램으로 메시지 주고 받기와 루틴을 함께 씁니다.

한국 법령 검색 MCP 연결하기
⏱ 10분중급Claude Desktop · 커스텀 커넥터
▼

법제처 Open API를 감싼 커스텀 커넥터를 붙여, 법령·판례·조례를 실시간으로 검색하고 원문으로 검증합니다. Claude가 지어내지 않고 근거를 인용하도록 환각을 잡아 줍니다. 이 커넥터는 아래 오픈소스 MCP를 그대로 씁니다.

korean-law-mcp 스킬 · github.com/chrisryugj/korean-law-mcp ›

핵심 스텝

  1. open.law.go.kr에 접속해 회원가입한 뒤, OPEN API > 활용 신청을 합니다. 그다음 마이페이지에서 OC 키를 확인합니다. OC 키는 보통 이메일 아이디와 같은 짧은 영문입니다.
  2. Claude Desktop을 열고 사용자 지정 → 커넥터 → 커스텀 커넥터 추가를 누릅니다.
  3. 이름은 알아보기 쉽게(예: Korean-Law) 적고, 주소 칸에 아래 MCP 주소를 붙입니다. 맨 뒤 YOUR_OC_KEY만 본인 OC 키로 바꿉니다.

복사용 MCP 주소

https://korean-law-mcp.fly.dev/mcp?oc=YOUR_OC_KEY

맨 뒤 YOUR_OC_KEY만 본인 OC 키로 바꿉니다. 예를 들어 OC가 honggildong이면 최종 주소는 https://korean-law-mcp.fly.dev/mcp?oc=honggildong이 됩니다. 주소는 반드시 https로 시작해야 합니다.

  1. 저장하면 잠시 뒤 search_law, get_law_text 같은 도구가 나타납니다. 이러면 연결된 것입니다.

텔레그램으로 메시지 주고 받기
⏱ 8분중급Claude Code터미널 전용
▼

텔레그램 봇을 Claude Code에 연결해, 분석 결과나 알림을 텔레그램으로 받고 봇에게 지시도 보냅니다. 봇 하나면 어떤 작업 결과든 휴대폰으로 받습니다. 위 기업 공시 분석과 묶어 매일 리포트를 받을 수 있습니다.

⚠️ 이 예제만 Claude Desktop이 아니라 터미널(CLI)입니다

/telegram:configure·/telegram:access는 Claude Code(터미널)에서만 실행됩니다. Claude Desktop 커넥터로는 붙일 수 없습니다. 먼저 터미널에서 아래 실행 명령으로 Claude Code를 켜면(텔레그램 채널이 함께 열립니다) 그 안에서 진행합니다.

핵심 스텝

  1. 텔레그램 앱에서 @BotFather를 찾아 대화를 엽니다. /newbot을 보내고 안내에 따라 봇 이름과 봇 아이디(끝이 _bot)를 정하면, 숫자로 시작하는 긴 토큰을 줍니다. 이 토큰을 복사해 둡니다.
  2. 터미널에서 아래 명령으로 Claude Code를 켭니다. --channels 옵션이 텔레그램 채널을 함께 엽니다. Claude Desktop이 아니라 터미널입니다.

Claude Code 실행 (텔레그램 채널 열기)

claude --channels plugin:telegram@claude-plugins-official

텔레그램을 쓰려면 Claude Code를 켤 때마다 이 명령으로 실행합니다. claude만 입력하면 텔레그램 채널이 열리지 않습니다.

  1. 아래 명령처럼 /telegram:configure 뒤에 1번에서 받은 토큰을 붙여 실행합니다.

토큰 등록 (숫자로 시작하는 봇 토큰을 뒤에 붙입니다)

/telegram:configure 8012345678:AAE1a2b3c4d5e6f_예시토큰

뒤의 8012345678:AAE...는 예시입니다. BotFather가 준 본인 토큰으로 바꿔 붙여넣습니다. 토큰은 숫자로 시작해 콜론(:) 뒤에 영문이 이어지는 긴 문자열입니다.

  1. 내 텔레그램에서 방금 만든 봇을 찾아 아무 메시지나 한 번 보냅니다. 그러면 페어링 코드(짧은 숫자)가 생깁니다.
  2. 아래 명령처럼 /telegram:access pair 뒤에 그 페어링 코드를 넣어 실행하면, 나만 이 봇을 쓰도록 허용됩니다.

접근 허용 (pair 뒤에 봇에서 받은 페어링 코드를 넣습니다)

/telegram:access pair 482913

뒤의 482913은 예시입니다. 4번에서 받은 본인 페어링 코드로 바꿉니다. 이렇게 나를 허용해 두면, 다른 사람이 봇에게 메시지를 보내도 나만 Claude에 연결됩니다.

  1. 봇에게 ping이라고 보냈을 때 Claude Code 화면에 그 메시지가 도착하면 연결이 끝난 것입니다.

봇 토큰은 비밀번호와 같습니다. 공개된 곳에 올리지 않습니다. 토큰은 내 컴퓨터의 ~/.claude/ 안에만 저장됩니다. 혹시 노출되면 BotFather에서 /revoke로 새 토큰을 다시 받습니다.

주의 사항

각 도구의 화면 구성과 메뉴 이름은 업데이트에 따라 바뀔 수 있습니다. 외부로 메시지나 데이터를 보내는 연결은 되돌리기 어려우니, 민감한 정보를 다룰 때는 사람이 한 번 확인하는 절차를 두고 사내 보안 정책을 함께 확인하시기 바랍니다.
