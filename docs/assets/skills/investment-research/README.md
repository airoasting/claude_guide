# 기업·산업 투자 리서치

특정 기업이나 산업을 공시·실적 같은 1차 자료로 정리해, 매수·매도 추천이 아닌 관점으로 마무리하는 리서치를 만든다.

## 무엇을 하나

회사명이나 산업을 주면 회사 모드와 산업 모드를 자동으로 판단한다. 웹 검색으로 공시·IR·10-K·실적 발표 같은 최신 자료를 모아 정해진 섹션 구조로 수치와 사실을 정리한다. 모든 증감은 절대값과 비율을 함께 적고, 확인할 수 없는 수치는 추정하지 않고 그대로 표시한다.

## 언제 쓰나

- "테슬라 분석해줘", "ENPH 어때?", "원자력 발전 밸류체인 조사해줘" 같은 요청
- 종목이나 산업을 투자 관점에서 빠르게 파악하고 싶을 때

## 입력

- 회사 모드: 기업명 또는 티커
- 산업 모드: 산업이나 시장 이름

필요하면 경영진·거버넌스나 특정 이슈를 중점으로 지정할 수 있다.

## 산출물

11개 섹션의 텍스트 리포트로 채팅에 출력한다.

- 회사 모드: 개요, 시장 규모, 성장 동력, 경쟁력, 경쟁사, 8분기 실적, 현금흐름, 유상증자 리스크, 경영진(요청 시), 최근 이슈, 종합 판단
- 산업 모드: 개요, 시장 규모, 성장 동력, 최근 이슈, 종합 판단

종합 판단은 별점과 모니터링 포인트로 마무리한다.

## 구성

- `SKILL.md` 발동 조건·워크플로우·출력 템플릿
- `references/report-template.md` 섹션별 상세 스펙
- `references/worked-example.md` 완성 예시
- `sample.html` 예시 리포트

## 이런 건 하지 않는다

- 매매 실행이나 포트폴리오 배분 조언
- "내가 사야 해?" 식 개인 투자 권유
- HTML 리포트·대시보드·파일 산출물 (dart 스킬 영역)

한국 상장사는 DART, 미국 상장사는 SEC 자료를 우선한다.
