# 메뉴개발 사이클 — 스테이지 전체 스펙 (v2.0)

각 스테이지의 목적·트리거·핵심액션·완료기준·**체크리스트(✓ 점검표)**·페르소나를 정의한다.
진행 시 해당 스테이지 섹션을 참조한다.

> v2.0 변경점
> - 각 스테이지에 **합격 판정용 체크리스트(✓/✗로 명확히 채점 가능한 항목)** 추가.
> - **위생·안전(HACCP·알레르기)** 점검을 2-1·2-2·4-1·4-2·5-1에 통합.
> - **관능평가 정량 시트**(5점 척도) 3-1에 추가.
> - **6단계 출시 후 점검(6-1)** 신설.
> - 모든 스테이지 페르소나 **최소 5인**.

체크리스트 표기 규칙: 각 항목은 `[ ]` 미충족 / `[x]` 충족 으로 판정하며, **모든 필수 항목이 충족(또는 사유 기록)되어야 완료기준을 만족**한다.

## 1-1 컨셉기획·시장조사

**페르소나**: 메뉴개발팀장 / 마케팅팀장 / 기획팀장 / 고객 / 현장 조리장

| 항목 | 내용 |
|---|---|
| 목적 | 시장 수요·브랜드 방향에 맞는 신메뉴 컨셉을 설정해 개발 단계의 실패 리스크를 사전 차단 |
| 트리거 | 시즌 메뉴 기획 주기 도래 / 신메뉴 요청 / 경쟁사·트렌드 변화 포착 |
| 핵심액션 | ① 시장·트렌드 조사 → ② 경쟁사 분석 → ③ 타깃·가격대 설정 → ④ 컨셉 후보 도출 |
| 완료기준 | 컨셉 후보 1~3안 + 목표 원가율·판매가 방향 정의 완료 |

**✓ 완료 체크리스트**
- [ ] 최근 6~12개월 트렌드·경쟁사 자료를 Tier 1~3으로 분류해 3건 이상 확보
- [ ] 목표 고객층·식사 시간대(점심/저녁/단품/정식) 정의
- [ ] 목표 판매가 + 목표 원가율 가이드라인 수치로 설정
- [ ] 기존 메뉴와의 중복·잠식(카니발리제이션) 여부 검토 결과 기록
- [ ] 핵심 식재료의 기존 매입 품목 연계 가능성 확인
- [ ] 계절성·공급 안정성 등 식재료 리스크 가설 1건 이상 명시
- [ ] 컨셉 후보별 한 줄 포지셔닝 문장 작성

**원칙**: 기획 단계에서 "무엇이 잘못될 수 있는가"라는 리스크 가설을 미리 세워두면 이후 단계의 의사결정이 빨라지고 폐기 비용이 줄어듭니다.

## 2-1 시제품 개발

**페르소나**: 메뉴개발팀장 / 메뉴개발팀원 / 현장 조리장 / 물류팀장 / 품질안전관리자

| 항목 | 내용 |
|---|---|
| 목적 | 컨셉을 실제 조리 가능한 시제품으로 구현하고 맛·조리난이도·재현성을 확보 |
| 트리거 | 컨셉안 확정 |
| 핵심액션 | ① 1차 레시피 작성 → ② 시제품 조리 → ③ 자체 시식·보완 → ④ 반복 개선 |
| 완료기준 | 매장 재현이 가능한 시제품 + 1차 레시피 초안 확보 |

**✓ 완료 체크리스트**
- [ ] 1차 레시피 작성 (재료·계량·순서를 숫자로 기록)
- [ ] 시제품 조리 후 자체 시식 (맛·식감·향·외관 기록)
- [ ] 조리 시간·난이도 측정 기록
- [ ] n차 보완 개선 이력 기록
- [ ] 매장 설비·인력 수준에서 재현 가능성 검토
- [ ] **알레르기 유발 원료(22종) 1차 식별·기록** ★
- [ ] **핵심 위해요소(생물학적·화학적·물리적) 발생 지점 1차 점검** ★
- [ ] 손맛·화력 의존 공정을 매장 표준 가능한 방식으로 치환

**재현성 주의**: 손맛·화력에 의존한 레시피는 매장에서 편차가 커집니다. 처음부터 매장 조건(설비·인력·시간)을 전제로 개발하세요.

## 2-2 원가·수율 검토

**페르소나**: 메뉴개발팀장 / 메뉴개발팀원 / 물류팀장 / 재무·원가담당 / 품질안전관리자

| 항목 | 내용 |
|---|---|
| 목적 | 식재료 원가율이 목표치를 충족하는지 검증하고 수율·로스를 사전 파악 |
| 트리거 | 시제품 1차 완성 |
| 핵심액션 | ① BOM 작성 → ② 수율 측정 → ③ 원가율 산정 → ④ 판매가·마진 시뮬레이션 |
| 완료기준 | 목표 원가율 이내 충족 + BOM 확정 |

**✓ 완료 체크리스트**
- [ ] BOM 작성 (1인분 기준 전 재료·분량·단가, 실제 조사 단가 기반)
- [ ] 가식부 수율(loss) 반영해 실사용량 산정
- [ ] 원가율 = 식재료 원가 ÷ 판매가(%) 산정, 목표치 충족 확인
- [ ] 판매가·구성 변경 시 원가율 변화 시뮬레이션 1건 이상
- [ ] **포장재·소모품 원가 별도 반영** ★
- [ ] **목표 마진·손익분기 판매량 산정** ★
- [ ] **보관 조건(냉장/냉동/상온)·소비기한에 따른 로스율 가정 기록** ★
- [ ] 운영비(임대료·인건비·공과금)는 별도 손익 영역으로 분리 명시

**수율 누락 주의**: 구매 단가만으로 원가율을 잡으면 손질 로스가 빠져 실제 원가가 과소 추정됩니다. 가식부 수율을 반드시 반영하세요.

## 3-1 경영진 시연·평가

**페르소나**: 메뉴개발팀장 / 마케팅팀장 / 기획팀장 / 성장운영본부장 / 대표 / 고객

| 항목 | 내용 |
|---|---|
| 목적 | 완성 시제품을 경영진에게 정식 시연하고 맛·외관·사업성 판단을 받아 도입 근거를 확보 |
| 트리거 | 시제품 + 원가 검토 완료 → 경영진 시연 일정 확정 |
| 핵심액션 | ① 시연 세팅(매장 제공 형태) → ② 관능 정량 평가 → ③ 사업성·운영 적합성 검토 → ④ 평가·보완 의견 취합 |
| 완료기준 | 경영진 평가 의견 취합 + 보완 항목 정리 완료 |

**관능 평가 정량 시트 (★ 5점 척도, 1=매우 미흡 · 5=매우 우수)**
| 평가 항목 | 척도(1~5) | 평가 기준 |
|---|---|---|
| 맛(간·감칠맛·균형) | ☐ | 목표 풍미 도달 여부 |
| 식감·텍스처 | ☐ | 목표 텍스처 일치 |
| 향 | ☐ | 식욕 자극·이취 없음 |
| 외관·플레이팅 | ☐ | 사진·실물 일치, 양감 |
| 가격 수용도 | ☐ | 판매가 납득 가능성 |
| 재구매 의향 | ☐ | 다시 주문할 의향 |

**✓ 완료 체크리스트**
- [ ] 실제 매장 제공 형태(그릇·양·온도)로 시연
- [ ] 조리 시작~제공까지 소요 시간 측정·공유
- [ ] 원가·판매가·예상 마진 자료 동반 제시
- [ ] 관능 평가 정량 시트 작성(항목별 점수 + 평균)
- [ ] 경영진 평가 의견·결정 사항 기록
- [ ] 보완 필요 항목과 우선순위 정리

**참고**: 시연은 시제품 개발과 분리된 별도 자리입니다. 개발 단계의 자체 시식으로 완성도를 충분히 끌어올린 뒤 경영진 시연에 올리세요.

## 3-2 도입 결정 게이트

**페르소나**: 메뉴개발팀장 / 마케팅팀장 / 기획팀장 / 성장운영본부장 / 대표 / 재무·원가담당

| 항목 | 내용 |
|---|---|
| 목적 | 경영진 시연 결과로 도입·보완·폐기를 판정해 검증되지 않은 메뉴의 매장 도입을 최종 차단 |
| 트리거 | 경영진 시연·평가 완료 |
| 핵심액션 | ① 경영진 평가 종합 → ② 원가·운영·위생 적합성 재검토 → ③ 도입/보완/폐기 결정 |
| 완료기준 | 경영진 도입 승인 또는 보완·폐기 결정 기록 |

**판정 기준**
| 판정 | 조건 |
|---|---|
| 도입 ✅ | 관능·원가·운영·위생 적합성 모두 기준 충족 |
| 보완 🔄 | 방향은 맞으나 일부 항목 미달 → 2단계 회귀 |
| 폐기 ❌ | 컨셉·수익성 근본 문제 → 종료 |

**✓ 게이트 판정 체크리스트**
- [ ] 관능 평가 평균이 합격선 이상
- [ ] 원가율·마진 목표 충족
- [ ] 운영(조리시간·동선·보관) 적합성 확인
- [ ] 위생·표시(알레르기·HACCP) 치명 결함 없음
- [ ] 판정 결과(도입/보완/폐기)와 사유 명문화

**게이트 무력화 주의**: "일단 내보내고 고치자"는 매장 혼선·재교육·클레임 비용을 키웁니다. 게이트는 형식이 아니라 실제 stop 권한이어야 합니다.

## 4-1 레시피 표준화

**페르소나**: 메뉴개발팀장 / 메뉴개발팀원 / 현장 조리장 / 물류팀장 / 품질안전관리자

| 항목 | 내용 |
|---|---|
| 목적 | 매장 간 품질 편차를 제거해 누가 조리해도 동일한 품질이 나오도록 규격화 |
| 트리거 | 도입 승인 |
| 핵심액션 | ① 계량 표준화 → ② 분량·중량 규격화 → ③ 조리 순서 표준화 → ④ 포션·플레이팅 규격 확정 |
| 완료기준 | 계량·중량·순서·포션이 명시된 표준 레시피서 확정 |

**✓ 완료 체크리스트**
- [ ] 모든 재료를 "적당히"가 아닌 g·ml·개수로 명시
- [ ] 1인분/세트 기준 분량·중량 규격 확정
- [ ] 조리 순서를 전처리·조리·플레이팅 단계로 분해
- [ ] 포션 도구·용기 규격 지정으로 양 편차 제거
- [ ] 배치 작업·사전 준비(전처리) 기준 포함
- [ ] **중점관리공정(CCP) 지정: 가열 중심온도·시간 기준 명시** ★
- [ ] **보관 온도·소비기한·재가열 기준 명시** ★
- [ ] **교차오염 방지 동선·기구 분리 기준 명시** ★

**원칙**: 작업 효율과 보관 표준(배치 계량·정리 방식 등)을 레시피와 함께 규격화하면 매장의 운영 부담이 줄어듭니다.

## 4-2 조리 매뉴얼 문서화

**페르소나**: 메뉴개발팀장 / 메뉴개발팀원 / 현장 조리장 / 운영본부장 / 품질안전관리자

| 항목 | 내용 |
|---|---|
| 목적 | 표준 레시피를 매장 전달용 문서로 변환해 교육·점검의 기준 문서를 확보 |
| 트리거 | 표준 레시피 확정 |
| 핵심액션 | ① 조리 매뉴얼 작성 → ② BOM·원가표 정리 → ③ 조리 사진·영상 가이드 → ④ 위생·표시 문서화 |
| 완료기준 | 매장 배포용 매뉴얼 패키지 완성 |

**✓ 완료 체크리스트**
- [ ] 조리 매뉴얼 (순서·계량·플레이팅·주의사항)
- [ ] BOM·원가표 (발주·재고 연계)
- [ ] 핵심 공정 주의사항 및 품질 기준
- [ ] 헷갈리기 쉬운 공정 위주 사진·영상 가이드 안내
- [ ] **알레르기 표시 정보(메뉴판·POS용) 문서화** ★
- [ ] **CCP 점검표·보관 온도 기록지 양식 포함** ★
- [ ] 서식·표기(용어·단위) 통일

**서식·표기 통일**: 글보다 "보고 따라 하는" 사진·영상이 매장 재현율을 높입니다.

## 5-1 매장 교육

**페르소나**: 메뉴개발팀장 / 현장 조리장 / 운영본부장 / 영업팀장 / 품질안전관리자

| 항목 | 내용 |
|---|---|
| 목적 | 매장 직원이 표준대로 조리할 수 있도록 현장 교육·실습 진행 |

| 핵심액션 | ① 교육 일정 협의 → ② 현장 시연 → ③ 직원 실습·피드백 → ④ Q&A → ⑤ 위생 교육 |
| 완료기준 | 교육 이수 + 매장 자체 재현 확인 |

**✓ 완료 체크리스트**
- [ ] 매장 일정 협의 후 방문/집합 교육 시행
- [ ] 표준 레시피대로 시연
- [ ] 직원 직접 실습 → 결과물 피드백
- [ ] 주로 나오는 질문·실수 정리(FAQ)
- [ ] **위생·알레르기·CCP 점검 교육 포함** ★
- [ ] 매장 자체 재현 결과물 합격 확인
- [ ] 교육 이수 명단·일자 기록

**원칙**: 신메뉴 도입·매장 형태 변경은 "문서 배포 + 현장 교육"을 한 세트로 처리해야 정착률이 높습니다.

**페르소나**: 메뉴개발팀장 / 영업팀장 / 현장 조리장 / 운영본부장 / 성장운영본부장

| 항목 | 내용 |

| 목적 | 실제 판매를 개시하고 대상 매장 전체에 메뉴를 반영 |

| 핵심액션 | ① 메뉴판·POS 등록 → ② 매뉴얼 배포 → ③ 매장 공지 → ④ 발주 연계 → ⑤ 초기 판매 점검 |
| 완료기준 | 판매 개시 + 대상 전 매장 반영 완료 |

**✓ 완료 체크리스트**
- [ ] 메뉴판·POS·키오스크 상품 등록
- [ ] **메뉴판·POS 알레르기 표시 반영** ★
- [ ] 매뉴얼·BOM 매장 배포
- [ ] 매장 공지 (판매 시작일·가격·구성) — 단일 창구 일원화
- [ ] 발주 품목·재고 연계 확인
- [ ] 도입 직후 초기 판매·이상 여부 점검 계획 수립

**원칙**: 도입일·가격·구성은 단일 커뮤니케이션 창구로 일원화해 매장별 혼선을 막습니다.
