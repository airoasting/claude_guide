# PoC Case 02 — 상담자 #123 (모바일 앱 입력)

## 입력 데이터 (요약)
- 연령 미입력, 세종 거주, 배우자 있음, 가족 동의 ○
- **자기자본 3천 / 즉시현금 2천 / 대출 계획 없음 / 기존 부채 1억**
- **6개월 예비자금 6천 (자기자본보다 큼 — 응답 모순)**
- 기대 월수익 500만, 회수 기대 12개월
- **외식·서비스업 10년 경력**, 직원 관리 ○, 전일 직접운영, 전업
- 후보 점포 없음, 상권 인식 없음
- 매뉴얼 전적 준수, SNS 직접 관리, 슈퍼바이저 환영
- 실패 대비책 없음

## 페르소나별 점수
| 페르소나 | 점수 |
|---|---|
| 레드 (리스크) | 2.0 / 10 |
| 골드 (운영) | 7.5 / 10 |
| 블루 (상권) | N/A |
| 화이트 (브랜드) | 7.0 / 10 |
| **종합** | **5.5 / 10** (N/A 제외) |

## 거부권
- **레드**: 자기자본 3천 + 부채 1억 + 응답 모순(자본<예비금)

## 최종 판정
**부적합** (단, 응답 모순 해소 시 **보류-개선조건부** 격상 가능)

## 8점 도달 로드맵 (요약)
- [레드 2.0 → 8.0] 응답 모순 해명 + 자기자본 6천 이상 확보 + 부채 월 상환액 명시 + 가맹비 자금 출처 확정
- [골드 7.5 → 8.0] 10년 경력의 구체적 업종·포지션 확인 + 조리 사전교육
- [블루 N/A → 8.0] 희망 행정동 명시 + 직접 답사 + 경쟁 매장 3곳 인지
- [화이트 7.0 → 8.0] 수익 외 선택 동기 명료화 + 경쟁 브랜드 비교 내용 정리

## 검증된 시스템 기능
- **응답 모순 자동 감지**: 자기자본 3천 vs 예비자금 6천의 산술적 모순을 분석 단계에서 포착, 재상담 최우선 질문으로 격상
- 거부권 + 점수 분리 표기: 골드 7.5·화이트 7.0의 강점이 점수에 반영되되, 레드 거부권으로 최종 부적합 결정
- 보류-개선조건부 격상 시나리오 제시: 모순이 단순 오기로 판명되고 자본이 실제 6천 이상이면 판정 격상 가능 명시

## 시사점
"운영 준비는 거의 완벽한데 자금이 위태로움" 패턴. 케이스 1의 정반대 프로필. 응답 모순 한 줄을 통화 한 번으로 확인하면 부적합 → 보류-개선조건부 → 적합까지 단계적 격상 가능한 케이스로, 본사 입장에서 가장 적극적 재상담 유치가 필요한 후보.
