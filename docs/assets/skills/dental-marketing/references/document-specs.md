
## 3. `BrandLab_브랜드전략_제안서.docx` (STEP 6, 스크립트: `make_proposal.js`)

클라이언트 전달용 제안서. STEP 7의 5color 루프가 이 파일만 반복 개선한다.

### 구성

```
표지: "BrandLab 브랜드 전략 제안서"
      부제: "환자가 선택하는 병원을 만듭니다 — 브랜드 전략부터 수익 구조까지"
한 줄 요약 (Executive Summary)
  - "광고대행사는 트래픽을 만듭니다. BrandLab은 환자가 선택하는 이유를 만듭니다."
1. 원장님 병원이 처한 현실 (경쟁 현황 데이터)
2. 제안 서비스
   - 기본 채널 관리 (네이버 플레이스·블로그·인스타그램·AI 리뷰 관리)
   - BrandLab 전용 서비스 3종: 원장·병원 브랜딩 / AEO / 수익 구조 진단
3. 패키지 및 가격 (5개 패키지 명시)
   (A) 채널 스타터 / (B) 채널 그로스 / (C) 브랜드 그로스
   (D) 개원 런칭 / (E) 브랜드 전략
4. 실행 일정 (5단계 로드맵, Phase 2: 외국인 환자 기반 포함)
5. 왜 BrandLab인가 (차별성 4가지)
   - 브랜드 전략 + P&L 동시 설계
   - 프리미엄 브랜딩 DNA (루이까또즈·지스타로우·바이파)
   - AEO·디지털 선점 기회
   - 첫 달 성과 보장
6. 다음 단계 (CTA + 연락처)
```

### 작성 원칙 (초안 생성 시 반드시 준수 — 5color 합격선의 사전 조건)

- BrandLab 포지셔닝 언어를 쓴다: "홍보 대행사"가 아닌 "브랜드 전략 파트너".
- 2인칭 톤을 쓴다: "강남 치과 40%가..." → "원장님 병원 주변 치과 중...".
- 가격을 명확히 표기한다: "협의"·"별도 문의" 없이 패키지별 상한·하한 금액 범위를 명시한다.
- P&L 관점을 강조한다: 광고비 효율이 아닌 수익 구조 관점의 언어를 쓴다.
- 의료광고법 준수를 명시한다: 과장 광고 없는 신뢰 기반 콘텐츠 원칙.
- AEO·브랜딩 효과 발현 시점을 구체화한다: "6~12개월 후 AI 인용 가시화".
- Phase 2 글로벌 비전을 포함한다: 외국인 환자 유치 기반 구축 로드맵.
- CTA를 명확히 한다: 상담 신청 경로와 무료 진단 제안.
